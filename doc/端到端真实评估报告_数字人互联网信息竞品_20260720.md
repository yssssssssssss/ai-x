### 技术改动

1. **`.env`**: 加 `TAVILY_API_KEY` + `TAVILY_BASE_URL=https://api.tavily.com`
2. **`RestJsonAdapter` 扩展 `body_inject`** — manifest 声明 `body_inject: {api_key: TAVILY_API_KEY}`,adapter 从 env 注入到 body(通用鉴权字段注入机制,未来其他 SaaS API 也复用)
3. **新建 `tools/tavily-web-search/`** — manifest + input.schema + output.schema
4. **tool-registry** 加登记(`adapter_type: rest_json`)
5. **2 个竞品 skill 的 `required_tools`** 加 tavily-web-search — 让 domain 预筛保留 + LLM 有意愿选
