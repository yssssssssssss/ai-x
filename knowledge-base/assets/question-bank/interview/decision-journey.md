# 访谈题库 · 决策与链路（decision-journey）

> **主题定义**：用户从"产生需求"到"最终下单"整条链路上的题——种草触发、搜索查找、关注信息与决策因素、正负向信号、因素排序、最终选择理由、多平台对比、场景差异。
> 与相邻主题的边界：纯"为什么想要/什么场景下用"归 [[needs-scenarios]]；纯"用完满不满意/哪里难用"归 [[experience-painpoints]]；对具体方案/功能的评价归 [[concept-test]]。
>
> **占位符约定**（本库题库统一）：题干里强绑定单次项目的名词用占位符，复用时替换即可——
> `[目标品类]`（如原题的"面点/亲子餐/药品"）· `[目标服务]`（服务类）· `[目标人群]`（如"孩子/老人"）· `[场景A]/[场景B]`（如"应急/囤货"）· `[本平台]`（如"京东"）· `[最终平台]/[其他平台]/[竞品平台]`（渠道对比）· `[店铺/频道]`（进店动线）。原始项目措辞见每条的「source / 变体」。

## 题组

### 决策链路还原（开放·实操还原）

- **主问**：请帮我还原一下您最近一次购买 `[目标品类]` 的过程——从产生"要买"的念头，到最终下单付款，中间都经历了哪些环节？
  - 追问：当时是什么情况/什么触发了这个念头？
  - 追问：去了哪些平台看？怎么挑到最后这款/这家的？中间有没有比价、犹豫、放弃又回来？
  - 追问：平时都是这么买的吗？还有哪些不一样的买法？哪种更多？
  - **实操提示**：可请用户共享屏幕、或打开订单记录，按真实操作习惯边演示边讲；有多平台对比习惯的，请一并还原。
  - source：qb4 门详差异化【路径还原】· qb7 面点拉新【整体决策链路还原】· qb2 亲子餐【场景回顾】
- **主问·念头触发（2026Q2）**：当时是什么具体的事/场景让您动了"要买"的念头（种草 / 急用 / 囤货 / 换新）？念头产生那一刻，脑子里第一个想到的购买渠道或品牌是什么？为什么？
  - source：电动牙刷【购买过程还原】· 驱虫【重点购买链路还原】
- **主问·第一次 vs 这次（2026Q2）**：这次购买和第一次购买，关注点有什么差别？有哪些是之前没在意、这次才在意的？
  - source：电动牙刷【购买情况回顾】
- **主问·阶段共创（2026Q2）**：这样听下来，您这次选购大概可以分成哪几个阶段？
  - **实操提示**：*不做引导，让阶段从用户经历中涌现（参考：需求/种草触发 → 初步了解品类 → 锁定具体款深入 → 多款对比 → 临近下单核实 → 买后）。*
  - source：运动鞋【选购历程重建&阶段锚定】
- **变体（差旅酒店预订全程·2026-07-08）**：请回忆最近一次差旅——去哪、什么目的？打开您订差旅酒店的 App，一起看您当时怎么挑到合适酒店的（可投屏）。
  - source：g4商旅 Part3【情景回溯 + 决策路径】
- **变体（服务购买还原·2026-07-08）**：请还原从"想买 `[目标服务]`"到挑选、到最终下单的完整经过；复购与首次挑选场景可分别还原。
  - source：g6服务频道 二【购买决策还原/购买行为还原】

### 种草与触发来源

- **主问**：当时怎么想到要买的？在哪里"被种草"过吗？
  - 追问：一般通过哪些渠道知道新的 `[目标品类]`？（短视频/直播、朋友推荐、线下看到、平台首页推荐、主动搜索……）
  - 追问：什么样的内容会吸引您？有没有被种草了但最后没买的？是什么打消了念头？
  - source：qb7【种草阶段】· qb2【触发情境】

### 种草攻略四层下探（做攻略过程·可复用范式·2026Q2）

- **主问**：您购买时做攻略了吗？大概用了多长时间？如何做的？请还原从"整体了解产品 → 缩小范围 → 确定产品"的过程。
  - 追问·阶段：有哪几个阶段的种草和攻略？（萌生兴趣 / 购买意愿 / 锁定具体产品）
  - 追问·渠道：在哪些渠道主动搜集信息（小红书/抖音/京东天猫等）？为什么想从多个渠道了解？不同渠道分别看什么？有没有配合用电商渠道？
  - 追问·内容：浏览了哪些信息？哪些加深了兴趣（功能/技术/使用效果/外观）？更倾向什么内容（素人的/横向对比的/展示细节的/生活场景的）？
  - 追问·形式：这些信息以什么形式呈现？哪种更打动您（产品测评/达人推荐/拆机/导购直播/vlog/专业科普/好物安利）？
  - **实操提示**：*母婴/户外等特殊场景可追问是否了解过专业平台、专业博主、专业群。*
  - source：电动牙刷【种草攻略】· 驱虫【信息收集与种草攻略】· 面点【种草阶段】
- **变体（服务/售前做功课·2026-07-08）**：买 `[目标服务]` 前会不会先做功课、找推荐？主要做哪方面功课、通过哪些渠道（熟人/物业推荐、自己搜、直接找平台）？哪些信息更有帮助，过程中有没有找不到想看内容、内容帮助不大的问题？
  - source：g6服务频道 二【信息收集/获取渠道/关注信息】· g7店铺化（进店做功课）

### 搜索与查找

- **主问**：您在平台上怎么找想要的 `[目标品类]`？会搜哪些关键词？
  - 追问：平台的分类/筛选好用吗？
  - source：qb2【搜索筛选】· qb7 · qb1 Part3-Q9（"一般逛/搜？搜什么？怎么筛选？"）
- **变体（服务/频道场景·2026-07-08）**：在 `[目标平台/频道]` 里怎么找想要的 `[目标品类/目标服务]`？除直接搜，还会用导航引导 / 找客服 / 看活动吗？知道有独立服务板块/频道吗、什么情况进入、会去找"全部服务"吗、入口好找吗？
  - source：g6服务频道【逛购习惯-寻找方式/入口路径】

### 多触点进店动机（电商店铺专用·穷举各入口·2026-07-08）

- **主问**：除了刚提到的入口，平常您还会从哪里进入店铺？分别什么场景进、进去想看/做什么？
  - **入口下探**（按用户涉及逐一展开）：首页推荐进店 / 搜索进店（结果页店卡 vs 商卡下店铺入口）/ 商详进店（想跳店铺首页还是商品页、为什么）/ 购物车·订单进店 / 关注进店（怎么发现并决定关注、想再进这家一般怎么找）/ 内容·直播进店。
  - source：g7店铺化 Part3【场景回溯/多触点进店动机差异-穷举各入口】

### 进店后浏览动线（店铺页/频道页专用·2026-07-08）

- **主问**：进入 `[店铺/频道]` 后您一般浏览哪些内容？先看什么、后看什么？（请投屏演示）
  - **模块下探**（按涉及展开）：首页（什么场景下滑、您认为应展示什么）/ 商品页（什么场景点进、关注上方新品·榜单·活动吗、符合预期吗）/ 发现页·内容 feeds（日常会关注吗、为什么、应展示什么）/ 会员页（什么场景主动入会、现有入会信息有吸引力吗、希望展示哪些）。
  - 追问（有目的 vs 无目的）：想找一款商品时怎么操作？无目的闲逛会点进哪些页面、看哪些内容？
  - source：g7店铺化 Part4【浏览动线/店内闲逛/信息获取】

### 关注信息与决策因素（核心题·可维度下探）

- **主问**：进入 `[店铺/商品]` 页面后，您首先关注哪些信息？为什么？其次呢？哪些信息会影响您最终是否下单？
  - **维度下探**（据用户提及展开，不逐条硬念）：
    - 优惠：关注哪些优惠信息？会主动找更多优惠吗？平台补贴 vs 优惠券更被哪个吸引？
    - 店铺：关注哪些店铺信息/标签？榜单、门店背景图/相册看不看？
    - 商品：怎么找想点的？关注价格/规格/成分/保质期哪些方面？
    - 评价：看不看评价？看哪类（店铺短评/评价页/商品短评）？希望看到什么内容？
    - 配送：关注配送费/距离/时间/自提吗？
    - 服务/会员：（如适用）问诊、会员权益等关注吗？
    - 判别标准（关键·2026Q2）：您如何判断这款"有效/好/安全/可靠"？看实验室数据、买家追评，还是博主实测视频？
    - 品牌/预算（2026Q2）：如何选品牌（进口/专业背景/口碑/科技/代言）？最初锁定预算区间吗？最后超预算没、因为什么（功能/品牌/外观）？
  - source：qb4【决策路径】各维度（外卖/超市/药品三场景重复，已合并）· qb2【外卖店铺·店铺关注信息】· qb7【挑选阶段&关注因素】· 电动牙刷/驱虫/宠物【关注因素逐维下探】（2026Q2）
- **变体（商超品类心智链路·2026-07-08）**：想买 `[某品类]` 时第一时间想到去哪买？最开始怎么知道这款商品/这个渠道的？是一开始就多在这买，还是逐渐偏向的？什么体验让您选择来这买？现在还跨渠道对比吗、对比什么？
  - source：g2京东超市 Part2【品类心智/购买契机/心智建立/对比行为】
- **变体（酒店结果页+详情页关注信息·2026-07-08）**：搜索结果页您怎么挑、优先关注哪些信息、什么信息吸引您进详情页（筛选标签够用吗、店铺/榜单标签/主图关注吗）？进详情页看哪些（相册/亮点设施/评价）？怎么找房型、房型对比关注哪些（楼层/大小/床型/办公设施）？
  - source：g4商旅 Part3【结果页/详情页/房型下探】

### 服务类决策因素（多主体·服务品类专用·2026-07-08）

- **主问**：您是怎么挑到最后这个 `[目标服务]` 的？主要会考虑哪些方面？
  - **维度下探**（按用户提及展开，不硬念，关注不同服务间的共性与差异）：
    - 平台：更看重品牌信任/平台保障/服务承诺/售后/客服？靠哪些内容判断？
    - 商家/服务商：是否关注是哪个商家提供？看重店铺资质/知名度/保障/覆盖范围？怎么判断资质？
    - 服务者（人）：是否关注具体服务者是谁？看重安全性/经验/人口属性/专业性/评价？怎么判断？
    - 服务产品：关注场景适配/服务内容/范围/预约时间灵活性？如何判断？
    - 价格：是否比价、怎么比？关注哪些优惠、喜欢哪种促销方式？
  - 追问（挑选难点）：有没有没头绪、不知道怎么挑的时候？当时怎么解决的？
  - 追问（决策风险）：挑选购买过程中，您最大的顾虑或担心是什么？
  - source：g6服务频道 二【决策因素-平台/商家/服务者/服务产品/价格/挑选难点/决策风险】

### 正向 / 负向信号

- **主问**：什么信息会让您觉得"这家/这款可以试试"？反过来，什么细节会让您立刻划走、觉得"不行"？
  - 追问（信任类）：必须看到堂食照片/连锁品牌/证书/后厨直播/买家实拍才敢下单吗？
  - source：qb2【搜索筛选·什么信息让你试试 / 什么细节让你划走】

### 关注因素排序 & 硬性门槛

- **主问**：请把您买 `[目标品类]` 时最看重的几个方面排个序。
  - 追问：有没有什么是"必须满足、不满足就一定不买"的？
  - source：qb4【关注点Top】· qb7【挑选阶段·按重要程度排序 / 必须满足的要求】

### 最终选择理由

- **主问**：最后您选了什么？为什么选它，而不是之前考虑过的其他几个？
  - 追问：价格、配送、包装、品牌，哪个对最终决定影响最大？
  - source：qb2【店铺选择 / 餐品选择】

### 最终渠道决策 + 京东专项分型追问（留存流失·可复用范式·2026Q2）

- **主问·决定性因素**：您最终选择在 `[最终平台]` 下单、而没选 `[其他平台]`，决定性因素是什么？
  - 追问：价格（最便宜/有券）？物流（着急用/次日达/小时达）？信任（正品）？服务（售后退换便利）？习惯（用惯了）？
  - **记录要点**：*平台转化的临门一脚是物流、价格还是信任。*
  - source：驱虫【最终购买渠道决策】· 电动牙刷【购买渠道选择】
- **主问·首选渠道心智**：提到买 `[目标品类]`，您最先想到哪个电商平台？为什么？它的优势是什么？您觉得买该品类的平台需要具备什么特征（货品/价格/服务）？
  - source：电动牙刷【首选渠道】· 奶粉【渠道选择】· 面点【平台认知与选择】
- **主问·京东专项分型追问**：您到 `[本平台]` 看相关产品了吗？（按分型择一追问）
  - 〔未浏览〕为什么没来看？为什么不会首先想到在 `[本平台]` 买 `[目标品类]`？您在 `[本平台]` 会买哪些品类？为什么那些品类在此买、这个品类却不买？
  - 〔浏览未购〕为什么最终没在这里购买？和其他平台相比有哪些不足？
  - 〔流失用户〕您去年在 `[本平台]` 买过、今年却选 `[其他平台]`，主要流失原因是什么？
  - 〔低频已购〕之后为什么没再来买？后来主要去哪个平台？那个平台比这里好在哪？什么情况下会再考虑？
  - source：电动牙刷/面点/奶粉/驱虫 京东分型追问（高度同构，折叠为 1 套分型模板）

### 多平台 / 多渠道对比

- **主问**：您会在哪些平台或渠道对比 `[目标品类]`？分别什么情况下用哪个？主要对比什么？
  - 追问：不同渠道在货品、价格、品质、服务上有明显差异吗？
  - source：qb4（多平台对比还原）· qb7【渠道选择·平台认知与选择】
- **变体（平台分工逐一投屏·2026-07-08）**：逐一就您提到的平台聊分工（邀请投屏）——什么情况来这个平台/进这个频道？频道内怎么找商品（逛/搜/活动）、找哪类？和频道外搜索的品类有区别吗、为什么这样分？
  - source：g2京东超市 Part3【平台分工-猫超/京东超市/盒马山姆/拼多多百亿超市】

### 跨平台比价还原（价格敏感链路·投屏实操·2026-07-08）

- **主问**：您日常购物会比价吗？主要买什么会比？在哪几个平台比？
  - **实操**：请就 `[目标品类]`（如食品饮料/纸品清洁/生鲜/小家电）投屏，还原您跨平台比较的完整过程。
  - 追问：您重点关注哪些信息来判断一个平台是不是便宜？`[本平台]` 哪些商品便宜、哪些不便宜，您是怎么感知到的？
  - source：g1特价版 Part3【平台比价-投屏还原/判断便宜的信息】

### 平台竞争与心智对比（三词投射·2026Q2）

- **主问**：提到买 `[目标品类]`，脑子里最先想到哪三个平台？用三个词形容在每个平台购物的感觉。
  - 追问：京东（正品/物流快/但贵？）· 淘宝天猫（种类全/小众品牌/但需筛选？）· 拼多多（便宜/但质量不稳？）· 抖音（看视频种草/冲动/价格优势？）
  - **记录要点**：*各平台差异化心智定位，用户买不同商品时如何分配平台角色。*
  - source：驱虫【各平台心智印象】· 面点【不同渠道优劣势】
- **主问·竞品美好体验借鉴**：在 `[竞品平台]` 购买时，有没有哪些功能/体验让您觉得"这个真不错"、比 `[本平台]` 更好？（内容/实测视频、互动/问大家、性价比/百亿补贴、专业客服）
  - source：驱虫【竞品平台美好体验借鉴】

### 复购链路与忠诚度（2026Q2）

- **主问**：如果上次买的效果很好，下次会怎么复购——直接找原订单，还是重新搜索比价？会换平台吗？
  - 追问：认准一个品牌/链接，还是每次重新研究？会优先在买过的平台复购吗？为什么？

## 使用说明

- **怎么用**：`generate-interview-guide` 组装"决策链路/购买决策"模块时优先取本文件，按 brief 替换占位符、按研究阶段裁剪题组；实物品类走"链路还原→关注因素→渠道"，服务品类补"服务类决策因素"，电商店铺研究补"多触点进店/进店动线"。
- **注意**：题组顺序即建议问序（链路还原破冰 → 逐环节深挖 → 排序收束 → 渠道/竞争/复购）；维度下探题不要逐条硬念，跟着用户提及走；务必保留访谈员意图注释（如"不主动提及询问，避免诱导"）——这是访谈题库比问卷更值钱之处。
- **自查**：落到具体提纲后过一遍 [[interview-forbidden-questions]]，剔除"让用户当专家/预言家"的问法。
