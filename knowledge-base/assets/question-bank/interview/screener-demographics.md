# 访谈题库 · 筛选与背景（screener-demographics）

> **主题定义**：甄别配额题、个人/家庭/工作/消费的基础信息、开场暖场自我介绍。只收"你是谁"。
> 与相邻主题边界：用户"平时怎么做/买"归 [[behavior-habits]]；"什么情况下才做"归 [[needs-scenarios]]。
> **占位符约定**：`[目标品类]`/`[目标人群]`/`[本平台]`/`[目标服务]`，复用时替换。

## 题组

### 暖场与自我介绍

- **主问**：先跟您认识一下——怎么称呼？哪里人？多大年纪？
  - 提示：开场先由访谈员做减负说明（今天像聊天、没有对错、真实感受即可、全程保密仅内部使用、需录音/开摄像头），再请受访者自我介绍。
  - source：qb1 Part1【自我介绍】· qb2 开场【受访者自我介绍】· qb4 开场语 · qb7 背景与规则介绍

### 开场破冰话术（可复用范式）

- **说明**：深访开场标准四步——欢迎与目的 → 真实想法鼓励 → 保密与录音 → 自我介绍；作为可复用开场模板整段调用。
  - 欢迎与目的：您好，非常感谢您参加此次访谈。本次主要想和您聊聊您在 `[目标品类]` 上的习惯和经验。您所讲的内容没有对错之分，分享最真实的想法就可以。
  - 真实想法鼓励（`[本平台]` 第一方研究特有）：关于 `[本平台]` 好的、不好的都可以说，不用因为我是工作人员就觉得不能说坏话，甚至更希望您多吐槽做得不好的地方，这样才能更好改进。
  - 保密与录音：访谈过程中我可能记录不全，会开启录制；您说的个人信息都会严格保密、仅内部使用，请放心。整个访谈大约 `[N]` 分钟，也会有同事线上旁听。您有什么问题吗？没有的话就正式开始。
  - 自我介绍：先介绍自己，再请用户自我介绍。
  - source：电动牙刷/宠物/驱虫/面点/运动鞋/PLUS/特价/奶粉 开场（8 份高度同构，折叠为 1 套开场范式）

### 个人与工作

- **主问**：您平时做什么工作？工作节奏怎么样，几点上下班、忙不忙？
  - 追问：工作之外的空闲时间多吗？一般怎么安排？出门多还是宅家多？
  - source：qb2 · qb4【生活习惯】· qb7【个人/工作】

### 家庭与同住

- **主问**：家里几口人一起住？有老人或 `[目标人群]`（孩子）吗？
  - 追问（有孩）：几个孩子？男孩女孩？多大了？主要谁照顾？
  - 追问（有老人）：老人多大年纪？多久见一次？在家主要负责什么？
  - 追问（同住对象·宠物·2026-07-08）：家里还有养宠物吗？什么宠物？（到家/养车等 `[目标服务]` 类研究关注居住结构与宠物）
  - source：qb2【家庭】· qb4【家庭居住】· qb7【家庭】· g6服务频道/g2京东超市【居住结构-孩子/老人/宠物】

### 消费与平台背景

- **主问**：平时常用哪些购物 App？哪些用得多、主要买什么？`[本平台]` 主要买什么？
  - 追问：`[目标品类]`（如厨房用品/做饭吃的）通常在哪买？线上 / 线下 / 外卖 / 超市 App？
  - 追问：和淘宝 / 拼多多 / 抖音比，`[本平台]` 占比大概多少？
  - source：qb7【消费情况】；平台占比追问补自 2026Q2 批量（PLUS未续费"购物习惯" · 运动鞋"网购经验"）
- **变体（平台使用背景速览·2026-07-08）**：您平时用 `[本平台]` 多吗？主要买什么？有特别常逛的页面/频道吗（如秒杀/直播）？（作背景速览，深挖平台内行为归 [[behavior-habits]]）
  - source：g5 AI改版【京东使用情况】· g8 AI购物【购物平台】

### 专业度与卷入度（特定研究）

- **主问**：您对 `[目标品类]` 的了解程度如何？会关注具体功能科技 / 参数 / 版型吗？平时有 `[相关行为]` 习惯吗？（频率 / 项目）
  - 提示：判断卷入度用，访谈员自然判断、不考问用户；深挖具体行为归 [[behavior-habits]]。
  - source：运动鞋"运动习惯/选鞋专业度" · 电动牙刷"口腔护理习惯"

### 饮食/使用现状速览（暖场承接）

- **主问**：工作日和周末的一日三餐分别怎么解决？有自己在家做饭的习惯吗？
  - 提示：作为从背景过渡到行为主题的桥接题；深挖归 [[behavior-habits]]。
  - source：qb4【饮食方案】· qb2【日常饮食】

### 甄别/配额表（招募期结构化采集）

- **说明**：招募或访谈开场用结构化表快速采基础变量，便于配额与后期分析。字段按项目定，常见：
  - 姓名 / 性别 / 年龄 / 城市 / `[目标人群]`相关（如孩子数量·年龄）/ `[目标品类]`使用场景。
  - source：qb2【基本信息表】
- **变体（配额维度扩展·2026-07-08）**：除常见字段外，可按项目加甄别维度——用户忠诚度 × 是否 `[本平台]` 主站用户（双端/独占）、`[本平台]` 是否主力渠道（购买频次最高）；注册时点 cohort（某节点前/后注册）；工作单位类型/性质（B 端/差旅报销主体研究）；住房与车产（租房/自有、房型、有无车辆及主要使用人——到家/养车类 `[目标服务]` 研究）。
  - source：g1特价版【配额：忠诚度×是否主站×注册前后】· g4商旅【工作单位类型/性质】· g6服务频道【住房/车情况】

### 招募配额条件（招募期）

- **说明**：招募筛选的配额设计范式（非访谈正文，供招募环节复用）：近 `[N]` 个月在 `[平台范围]` 购买过 `[目标品类]`；`[核心分型]` 均衡（如猫:狗=3:3、新手:老手、高线:低线）；`[关键条件]`（如单价 2000 元以上、异地购买经历、渠道分散覆盖直播+货架电商）。
  - source：宠物活体"用户配额"（近 3 月线上购买活体、猫狗 3:3、单价 2000+、异地优先、新手老手均匀）· 驱虫"访谈对象"

## 使用说明
- 怎么用：`generate-interview-guide` 组装开场/暖场模块时取本文件；按 brief 替换 `[目标人群]`/`[目标品类]`，删掉与本次无关的甄别字段。
- 注意：暖场题不宜多；减负说明必须在自我介绍前先讲。
- 自查：过 [[interview-forbidden-questions]]（背景题避免直接问敏感的收入/隐私，必要时改区间或"不便透露"兜底）。
