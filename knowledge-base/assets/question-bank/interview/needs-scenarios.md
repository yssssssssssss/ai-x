# 访谈题库 · 需求与场景（needs-scenarios）

> **主题定义**：触发场景、动机、"什么情况下会用/买"、不同场景的差异。收"何时/为何要"。
> 与相邻主题边界："平时怎么做"归 [[behavior-habits]]；"怎么一步步选和买"归 [[decision-journey]]。
> **占位符约定**：`[目标品类]`/`[目标服务]`/`[目标人群]`/`[本平台/目标频道]`/`[场景A]·[场景B]`/`[AI功能]`，复用时替换。

## 题组

### 典型场景枚举

- **主问**：您通常在什么情况下会用/买 `[目标品类]`？
  - 追问：最常出现的是哪一种？大概多久遇到一次？
  - 追问：还有哪些不那么常见但会发生的场景？
  - 追问：这些场景里通常都有谁一起？有没有专门为 `[目标人群]`（老人/孩子/宠物）准备的情况，更侧重什么？
  - source：qb2【典型场景】· qb4【典型场景&行为特征】· qb7【面点食用场景】· 驱虫【居住环境/使用区域】· 宠物【购买原因】
- **主问·变体（服务/到家类）**：您通常在什么情况下会用到 `[目标服务]`？最常见的是哪种，多久遇到一次？还有哪些不那么常见但会发生的场景？
  - source：g6服务频道【场景需求】· g3外卖月卡【外卖需求场景&特征】
- **主问·变体（逛购触发/非目的场景）**：除了有明确要买的东西时，还有什么情况下会打开 `[本平台/目标频道]`？有固定逛的时间点/契机吗？吸引您来逛的原因是什么（玩法/活动/内容）？
  - source：g6服务频道【服务平台/频道逛购场景-非目的场景/使用契机/目的动机】

### 动机与深层诉求

- **主问**：您为什么会想要 `[目标品类/某属性]`？（如"为什么想吃得营养"）
  - 追问（长期）：背后是想达到什么？（如保持状态/身形/免疫力…）
  - 追问（触发）：什么特殊时点会更想要一些？（如生病后/换季/熬夜后/加班后…）
  - source：qb1【营养动机】

### 需求激发与第一念头

- **主问**：当时是什么具体的事情或场景，让您动了"必须得买 `[目标品类]`"的念头？
  - 追问：是"种草"（看到别人推荐）、"急用"（突然发现需求爆发）还是"囤货"（家里用完了）？
  - 追问：念头产生那一刻，脑子里第一个想到的购买渠道或品牌是什么？为什么？
  - 提示：重点记录需求激发的场景类型（种草/急用/囤货）与首选心智渠道及原因。
  - source：驱虫【需求激发与第一念头】· 运动鞋【购买原因】· 宠物【购买原因/启动场景】
- **主问·启动场景**：还记得您是从哪里、通过什么方式第一次看到/想到的吗？（刷视频/刷推荐/看种草/看直播/搜索）当时是什么吸引了您？当时生活正处于什么阶段或场景，促使您决定？
  - source：宠物【需求场景还原】· 运动鞋【购买驱动】

### 场景下的关注差异

- **主问**：在 `[场景A]` 和 `[场景B]` 下用/买 `[目标品类]`，您关注的点有什么不一样？
  - 追问：不同场景下分别更在意什么？（品类/口感/配送/价格…）
  - source：qb2【不同场景关注因素】· qb4【场景差异】
- **主问·变体（差旅多场景）**：对比不同 `[场景A]/[场景B]`（如客户拜访/展会/培训/项目常驻；不同城市线级；单人住/同住），您对 `[目标品类]` 的偏好或关注点有什么不一样？
  - source：g4商旅【差旅酒店预定偏好-多维差异】· g3外卖月卡【外卖需求场景】

### 对象差异：给自己 vs 给他人

- **主问**：`[目标品类]` 单独给 `[目标人群]`（如孩子）用，和全家/给自己用，哪种更多？
  - 追问：什么情况只给 `[目标人群]`、什么情况全家一起？两者在选择上有哪些不一样、各自麻烦在哪、好处在哪？
  - 追问：家里还有谁也会给 `[目标人群]` 张罗？亲友之间会互相帮点/帮买吗？什么原因？
  - source：qb2【只给孩子点 VS 全家一起点】【给孩子点的家庭成员】

### 场景回顾（还原一次真实经历）

- **主问**：请回想您最近一次在 `[目标场景]` 下用/买 `[目标品类]` 的经历，跟我讲讲当时的情景。
  - 追问【触发】：当时是什么情况让您选择这么做，而不是别的方式？
  - 追问【角色】：是 `[目标人群]` 主动要求还是您决定？TA 怎么参与的？
  - 提示：可请用户打开订单/记录，找最近 1–2 次特意做的来还原（下单细节归 [[decision-journey]]）。
  - source：qb2【场景回顾-触发情境/角色分工】

### 转化场景设想（潜在用户）

- **主问**：请您设想一下，未来发生什么样的具体事情或变化，最有可能让您认真考虑买 `[目标品类]` 试试？
  - source：电动牙刷【潜力用户-转化场景】
- **主问·购买阻碍**：一直没尝试 `[目标品类]`，最主要的原因是什么？（觉得现有够用、价格高、担心不适应、觉得是智商税等，每个单独追问）
  - source：电动牙刷【潜力用户-购买阻碍】

### 需求触发情景还原（"什么情景下产生这个需求"）

- **主问**：请从最近 1 次印象深刻或最有代表性的经历聊起——您是在什么情景下产生需要 `[目标品类/目标服务]` 的想法的？
  - 追问（触发类型）：是定期计划、突发故障、季节更替，还是特定事件（如"过年大扫除""租房退租前"）？当时希望解决哪些问题？
  - 追问（连带需求）：当时同时还买了别的服务/商品吗？连带需求是自发产生的，还是被激发的？怎么被激发的？
  - 提示：与已有「场景回顾」互补——本题侧重"需求怎么冒出来的 + 连带激发"，下单细节归 [[decision-journey]]。
  - source：g6服务频道【需求场景还原/连带场景】· g4商旅【情景回溯】

### 使用场景与设施诉求（"用它来做什么"）

- **主问**：在 `[目标场景]` 下用 `[目标品类]` 时，除了核心用途，您通常还会用它做什么 / 用到哪些配套设施或服务？
  - 提示（按品类给下探锚点，不硬念）：如差旅酒店——房间办公 / 大堂见客户 / 开会 / 打印 / 熨烫 / 健身 / 吃饭等。
  - source：g4商旅【酒店使用情况-除过夜还做什么】

### AI 介入的需求场景（"什么情况下会想用 AI"）

- **主问**：通常在什么情况下您会想到要用 `[AI功能]` 来辅助？主要希望它帮您解决什么问题？
  - 提示（按购物链路给发散锚点，不限定）：购前灵感激发 / 商品搜索 / 评估决策 / 交易咨询 / 购买下单 / 购后阶段。
  - 追问（意图模糊 vs 明确）：想买但还没想好/不会挑时会先做什么？已经很明确想买某款时又会怎么用？
  - source：g8 AI购物【使用场景-分阶段锚点】· g5 AI改版【搜索-意图模糊/意图明确】

## 使用说明
- 怎么用：`generate-interview-guide` 组装"场景与需求"模块时取本文件；场景枚举题适合承接行为现状之后、进入决策深挖之前。
- 注意：场景题重在"让用户还原亲历"，不要问假设（"如果…你会不会"）——假设题归偏误自查剔除。
- 自查：过 [[interview-forbidden-questions]]。
