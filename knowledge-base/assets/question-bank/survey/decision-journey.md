# 问卷题库 · 决策与链路（decision-journey）

> **主题定义**：购买决策相关的量化题——关注因素、选店/选品偏好、信息展示优先级、筛选方式、优惠呈现。收"选和买时看什么、怎么选"。
> **占位符约定**：`[目标品类]`/`[目标人群]`/`[本平台]`，复用时替换。保留 **题型 / 选项池 / 逻辑**。

## 题组

### 关注因素（重要性）

- **【多选·选项随机】用 / 买 `[目标品类]` 时，您比较关注以下哪些方面？** — 关注因素池（口味/食材/健康/安全/营养/份量/价格/配送/口碑…，"其他"沉底）。
- 说明：可配 IPA / Kano / KDA 分析（见 [[satisfaction-analysis-models]]）；重要性排序可另设排序题。
- 变体（2026Q2·跨品类）：可复用下方"通用品类决策链路电池"的关注因素行——通用因素池外按品类替换专属池（奶粉配方/奶源/段数 · 电动牙刷清洁力/智能化 · 白酒香型/度数/口感 · 厨电能效/套装统一性 · 驱虫起效/安全性/广谱性）。
- **【多选】变体（逛购麻烦点·AI 研究痛点因素池）：在电商 App 逛购时，您较常遇到哪些让您麻烦 / 不满意的情况？** — 麻烦点池（推荐不感兴趣 / 搜索麻烦 / 难筛选 / 不确定实际效果 / 对比繁琐 / 看不懂参数 / 评价太多 / 不知如何买到低价 / 凑单麻烦 / 客服低效 / 物流异常 / 售后繁琐；"其他""没遇到"沉底）。
  - 说明：s5 中与"希望 AI 帮助的情况"成对出现（同一因素池问"现状麻烦"＋"想要 AI 帮"），是"痛点 → AI 需求"配对结构，需求侧见 [[concept-test]]。
- source：qb3 Q9 · qb6 挑选关注 · s5 AI动线 Q9（逛购麻烦点变体，+ Q10 AI 需求配对）

### 信息展示优先级 —— Kano 五档题组（可复用模式）

> **一个属性池 × 五种问法**测每条信息的 Kano 归类。原问卷对"店铺信息"和"商品信息"各跑一遍五问（qb5 Q7–Q11、Q12–Q16），**去重折叠为 1 个题组模式**，`[对象]` 换成 店铺 / 商品 / 其他即可复用。

- **同一属性池，依次问 5 题（均多选）**：
  1. 【必备】哪些信息"必须展示，不展示肯定不行"？
  2. 【期望】哪些"不是必须，但展示了更有帮助、不展示会影响下单"？
  3. 【魅力】哪些"展示了会很惊喜，但不展示也不影响下单"？
  4. 【无差异】哪些"展示不展示都无所谓、不太关注"？
  5. 【反向】哪些"展示了很多余，不展示更好"？
- **属性池（示例·`[对象]`=外卖店铺）**：店名 / 是否品牌连锁 / 是否新店 / 是否堂食店 / 24h / 跨天预定 / 开发票 / 饭卡 / 后厨直播 / 到店自取 / 评分 / 销量 / 榜单 / 店铺特色 / 回头客数 / 推荐菜 / 优惠 / 配送时长距离 / 起送与配送费 / 配送方式 / 食安保障 / 现炒 / 人均 / 短评价。
- **属性池（示例·`[对象]`=外卖商品）**：餐品名 / 图 / 所属店 / 已售 / 回头客推荐 / 近期下单数 / 好评率 / 短评价 / 食材 / 口味 / 分量 / 到手价 / 优惠 / 限购 / 招牌菜 / 平台专供 / 配送费·时长·距离。
- 用法：后 4 题选项池引用第 1 题（问卷里写"选项同 QX"）。
- 变体（关注因素→信息展示，与 Kano 互补）：另有一种"显示逻辑式"信息优先级测法——见下方「关注因素 → 分因素信息展示追问（显示逻辑 battery）」，对每个被选中的关注因素单独追问希望展示的信息；可作为 Kano 五档之外的并列测法，供 `generate-survey` 按题量二选一。source：s1商旅 Q5–Q18。
- source：qb5 Q7–Q11（店铺）+ Q12–Q16（商品），Kano 五档折叠

### 关注因素 → 分因素信息展示追问（显示逻辑 battery·可复用模式）

> 先用一道多选测"整体关注哪些因素"，再对**每个被选中的因素**追问"希望平台展示哪些信息"（显示逻辑 `因素题=该项时展示`）+ 一道开放补充。s1 对酒店的 **8 个因素各追一组**（Q6–Q18），**折叠为 1 个题组模式**，`[关注因素]` 替换即可复用到任何"多因素 × 信息展示"研究。

- **【多选·选项随机】主问：在 `[目标场景]` 下选 `[目标品类]` 时，您通常关注哪些因素？** — 因素池（示例·差旅酒店：连锁品牌 / 地理位置 / 房间环境 / 睡眠体验 / 办公商务 / 用餐 / 生活设施 / 酒店政策；"其他"兜底）。
- **追问模式（每个被选因素一组，显示逻辑＝选中该因素时展示）**：
  - **【多选】针对 `[关注因素]`，平台展示哪些信息会让您觉得更方便、更贴心？** — 该因素的信息点池 + "以上都不关注"。
  - **【文本】除上述，在 `[关注因素]` 方面您还希望展示哪些信息？** — 开放补充。
- **信息点池（示例·酒店各因素）**：地理位置→距办事地 / 机场火车站 / 地铁 / 接送；房间环境→面积 / 卫生 / 智能化 / 安静 / 通风 / 装修新旧；睡眠→床垫 / 枕头 / 遮光 / 隔音；办公→办公桌 / 插座 / 网络 / 会议室；用餐→含早 / 餐食丰富度 / 供餐时间 / 打包 / 夜宵 / 周边餐饮；生活设施→洗衣 / 健身泳池 / 熨烫；政策→退改 / 立即确认 / 延时退房 / 押金 / 开票。
- source：s1商旅 Q5–Q18（关注因素 + 8 维度信息展示追问，显示逻辑 battery 折叠）

### 抢券 / 用券行为链路（营销活动类专用）

> 营销活动 / 权益领取类问卷的行为链路：知晓渠道 → 抢券行为分型 → 抢到 / 没抢到后的复来意愿 → 时段偏好。`[目标活动]`/`[目标权益]` 替换即可复用。

- **【单选·选项随机】您最先通过哪个渠道知道 `[目标活动]` 的？** — 渠道池（站内 / 线下广告 / 内容平台 / 其他 App / 微信 / 熟人；"其他"沉底）。
- **【单选】关于近 N 天抢 `[目标权益]` 的行为，哪项符合您的实际？** — 行为分型（有需求才来抢 / 专门来抢 / 顺手抢）。
- **【单选 ×2】当天抢到了 / 没抢到，之后再来抢的意愿是？** — 意愿分型（有需求会来 / 有空就来 / 单次薅羊毛不再来）；抢到与没抢到各问一次。
- **【多选】您倾向一天中什么时段来抢券？** — 时段池（"无固定偏好"排他沉底）+ **【文本】** 偏好该时段的原因。
- 说明：s3 的"成功抢到的概率感知"（单题、绑活动机制）价值低，折叠进本题组备注，不单列入库。
- source：s3金券 Q6 / Q8 / Q10_1 / Q10_2 / Q11 / Q12

### 流失归因与召回（留存/流失·折叠范式）

> 生鲜流失、驱虫、电动牙刷(换回手动)、奶粉(渠道迁移) 四份同构，测"为何不在本平台买/为何流失→什么能召回你"。折叠为流失归因 + 召回举措两条模式；现有题库空白，独立成节。

- **【单选】以下哪种描述更符合您（近期未在 `[本平台]` 购买 `[目标品类]`）的情况？** — 流失类型池：某次体验差 / 多次体验差渐渐流失 / 需求变了平台满足不了 / 只买特定品近期不需要 / 主动转去别家更好更便宜 / 常住地作息变更配送不合适 / 改版后找不到入口 / 只是没想起来。
- source：生鲜流失 A1 · 电动牙刷 C6/C7(换回手动)
- **【多选·限N项·选项随机】那次/那些体验让您不再购买，主要问题是？** — 流失原因池（分层）：商品(品质/种类少/规格不符/缺货) · 价格(降价/促销复杂/优惠缩水/比别家贵) · 体验(搜索不准/详情不全) · 服务(配送延误/不新鲜破损/售后被拒/客服差/不上门)。
- source：生鲜流失 A2/A3 · 驱虫 C8/C9/C10 · 电动牙刷 C5/C7
- **【多选·选项随机】您没有到 `[本平台]` 浏览/购买 `[目标品类]` 的原因是？** — 未触达/未转化归因（印象中价格贵/种类少/不卖这类"小东西"/买过效果不好/需运费凑单/不知道活动/没会员/缺直播讲解/评价少/很少用本平台）。
- source：驱虫 C8(没来京东)/C9(来了没买)/C10(不是首选) · 电动牙刷 A8 · 厨电 Q11 · 奶粉 Q34c
- **【多选·选项随机】您目前使用的其他渠道，在哪些方面比 `[本平台]` 做得更好？** — 竞品优势池：种类丰富度/品质/价格/优惠频次力度/会员权益/送货速度/包装保鲜/配送灵活度/包邮门槛/操作简易/优惠券好领/售后。
- source：生鲜流失 A6
- **【多选·限N项】`[本平台]` 做出以下哪些改进，您会更愿意重新购买/召回？** — 召回举措池：品质更稳定/品类更丰富/更高品质选项/便捷选项/规格适配/促销更简单直降/专属会员折扣/新客老客券/配送更快/超时赔付/包装更好/售后省心/入口更好找/可指定配送时段。
- source：生鲜流失 A7 · 奶粉 Q35(京东牵引) · 电动牙刷 C8(重新考虑条件)

### 平台竞争与渠道迁移

> 奶粉、驱虫有完整的"多平台对比 + 渠道迁移契机 + 迁移原因"链路，可复用为 `[本平台]` vs `[竞品]` 的迁移分析模式；现有题库空白，独立成节。

- **【矩阵单选】最近1年，您在各渠道购买 `[目标品类]` 的频率变化？** — 大幅下降/小幅下降/没变化/小幅上升/大幅上升 × 各平台。
- source：驱虫 C7-2
- **【单选】近1年您购买 `[目标品类]` 的首选渠道发生过变化吗？** + **【单选】之前的首选渠道是？** + **【多选】迁移主要由什么触发？** — 迁移契机池：进入新阶段/紧急需求原渠道满足不了/被新渠道强促销吸引/被亲友种草/对原渠道一次严重不满/长期体验不佳/整体购物习惯改变。
- source：奶粉 Q31/Q32/Q33
