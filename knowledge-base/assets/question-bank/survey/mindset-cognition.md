# 问卷题库 · 心智与认知（mindset-cognition）

> **主题定义**：态度/观念分型、品牌偏好、平台/活动认知的量化题。收"怎么想/什么态度"。
> **占位符约定**：`[目标品类]`/`[目标人群]`/`[本平台]`/`[目标活动]`/`[关联子品牌]`/`[目标标签]`/`[AI功能]`，复用时替换。

## 题组

### 观念/态度分型

- **【单选】在您的观念中，"`[目标行为]`"（如给孩子点外卖）这件事最接近以下哪种描述？** — 态度分型选项（如 救急/偶尔奖励/已成常态…）。
- **【单选】下列哪种最符合您的 `[目标观念]`（如养育/消费）观念？** — 用于人群态度细分。
- source：qb3 Q24 / Q26

### 品牌偏好与尝新

- **【单选】您对品牌的偏好是？** — 如"只认固定品牌 / 愿意尝新"（尝新偏好，用于分型）。
- source：qb6 品类购买习惯（品牌偏好·尝新偏好）
- **变体（差旅品牌标准化偏好·态度侧·2026-07-08批）**：出差订酒店时，您是否有常住品牌？（若有，多因标准化/会员/信任）—— 作态度分型输入。
  - source：s1商旅（品牌偏好，主体在 decision 关注因素）

### 平台/活动认知

- **【单选】您是否听说过 `[本平台]` 的 `[目标活动]`（如"黑色星期五"）？** — 认知漏斗第一层。
- **【单选】您是否在 `[目标活动]` 中购买过商品？** / **【多选】购买过下列哪些品类？** — 认知→转化漏斗。
- source：qb6 黑五认知度
- **变体（活动知晓 benchmark·2026-07-08批）**：您知道以下哪些 `[本平台]` 的 `[目标品类]` 优惠活动？ — 活动清单（选项随机，含一个已知高知晓活动作可比 benchmark，"都不知道"沉底→可终止）。
  - source：s3金券 Q3（外卖优惠活动知晓，百亿补贴作 benchmark）

### 平台/活动心智五感评价（核心范式·高复用·2026Q2）

> 用"丰富/品质/实惠/服务/有趣"五感测平台与活动心智，每感配"低分下探原因"。可复用为 `[平台/活动]` 心智五感评价（可 related 满意度四模型/HEART）。

- **【矩阵量表】您对"`[平台/活动]` 的 `[目标品类]` 是 `[某感]` 的"这个说法的同意程度？** — 5级同意度（非常不同意→非常同意）× 各平台。五感维度：①丰富感（品类/品牌/款式）②品质感 ③实惠感 ④服务感 ⑤有趣感（活动特有）。
- **【多选·选项随机】您认为 `[平台]` 的 `[目标品类]` 不够 `[某感]` 的原因是？** — 各感低分下探池：
  - 不够丰富：品类不全 / 品牌不丰富 / 款式颜色IP不全 / 某价位段少 / 新品少 / 缺货 / 重复推荐。
  - 不够品质：山寨假货多 / 自营官方店少 / 参数细节不全 / 介绍夸大 / 缺品质承诺 / 差评多。
  - 不够实惠：同款比别家贵 / 价格波动 / 促销少 / 优惠门槛高 / 促销复杂 / 折扣不真实。
  - 不够服务好：客服慢 / 解决力弱 / 退换货复杂 / 审核慢 / 送货慢 / 包装弱 / 说明难找。
  - 不够有趣：商品无趣缺IP联名 / 主题无聊 / 形式老套 / 卖货感强。
- source：文具 A5–A8·Q7/Q9/Q11/Q13（平台四感+下探）· B8/B10/B12·B9/B11/B13（活动三感+下探）（2026Q2）

### 平台/品类认知漏斗（2026Q2）

- **【填空·无提示】想到购买 `[目标品类]`，您首先想到哪个平台/渠道？** + **【填空】还会想到哪些？** — 无提示第一提及 + 联想（心智强度）。
- **【多选·提示后】以下平台，您知道哪些可以购买 `[目标品类]`？** — 提示后知名度。
- **【填空/多选】想到 `[某需求]`，您会想到哪些主题活动？** + **【多选】您听过以下哪些活动？** — 活动认知（主动想起 + 提示后）。
- **【单选】您是否知道/听说过 `[某功能/权益/概念]`？** — 功能/权益认知第一层（如 AI清单购、免申购费、大屏卡）。
- source：文具 A1/Q2/A2·B1–B5·B21 · 驱虫 C1/C2/C3 · PLUS D1/E9（2026Q2）

### 品类观念量表 & 品牌态度（2026Q2·衔接品牌偏好与尝新）

- **【矩阵量表】您对以下关于 `[目标品类]` 的说法是否认同？** — 认同度量表，说法池为品类刻板印象/观念（如"低度酒=减配""光瓶酒=勾兑""光瓶酒更实在"）。
- **【矩阵单选/单选】您对 `[目标品类]` 品牌的态度/偏好是？** — 品牌重要性：必须知名品牌 / 有品牌就行 / 效果好就行 / 无所谓。
- **【单选】您如何判断 `[某属性]`（如安全性/口碑）？** — 判断依据池（认知信号来源）。
- source：低度酒 A12 · 光瓶酒 A10 · 驱虫 A12/A7/A8 · 奶粉 品牌偏好/Q25/Q26（2026Q2）

### 活动对品类印象的影响（2026Q2）

- **【矩阵多选/多选】了解到 `[活动]` 有 `[某品类]` 后，对您购买该品类有哪些影响？** — 影响分层（无影响 / 来看了没买 / 考虑了 / 直接买了）。
- **【多选】参与 `[活动]` 后，您对 `[平台某品类]` 的印象有哪些改变？** — 印象改变池（更实惠 / 更潮 / 更专业 / 更丰富 / IP多 / 新奇 / 没变）。可复用为 `[活动]×[品类]` 印象提升模式。
- source：文具 C1–C8（2026Q2）

### 满意度-NPS 度量组（评价侧·核心增量·2026Q2）

> 与 [[experience-painpoints]] 的"不满原因下探"配合使用；NPS/CSAT 标准口径见 [[standardized-ux-scales]]，避免重复造轮子。

- **【矩阵单选】请对您在 `[各平台/各权益]` 的整体经历/各维度进行满意度评价** — 1–5 分满意度矩阵（非常不满意→非常满意）× 对象/维度。
- **【矩阵量表·NPS】您有多大可能向亲友推荐 `[平台/活动]`？** — 0–10 分 NPS × 各对象。
- **【多选】您愿意/不愿意推荐 `[平台/活动]` 的原因是？** — 双向原因池（推荐：商品丰富/质量好/价格低/促销多/物流好/售后好/客服好/推荐精准/内容吸引；不推荐：对应负向）。
- **【矩阵单选】请对 `[平台/活动]` 的以下维度评价**（价格/折扣力度/品质/丰富度/趣味性/服务保障）— 多维度满意度矩阵。
- source：文具 Q16/B14·Q18/B17·Q19/Q20/B18–B19·C2 · PLUS A4/A5/A10（2026Q2）

### 店铺类型总体态度（电商店铺专用·2026-07-08批）

- **【单选】您在 `[本平台]` 选购时，对各类型店铺的态度是？** — 只考虑自营/旗舰店 / 也会考虑第三方卖家 / 不关注店铺类型。
- source：s2店铺 Q1

### 店铺标签态度验证 —— 「态度 + 未正向原因」二段式 battery（可复用模式·2026-07-08批）

> **一个标签 × 两问**测其心智接受度。对 `大牌同厂/源头工厂/百元店/折扣仓/全网金牌卖家/品质质检/潮流好货/京喜好店` 等 **8+ 个标签各跑一遍**（态度单选 + 未正向原因多选，后者显示逻辑 `态度=中立/负向时展示`）。**去重折叠为 1 个题组模式**，`[目标标签]` 替换即可复用。

- **同一标签，依次问 2 题**：
  1. 【单选·态度】当您在 `[本平台]` 选购时，看到店铺被打上"`[目标标签]`"的标签，您的态度是？ — 比较正向，更愿信任 / 保持中立，不影响判断 / 比较负向，不看好。
  2. 【多选·未正向原因】您对"`[目标标签]`"标签未持正向态度的主要原因？ — 原因池（噱头怀疑/质量顾虑/售后担忧/含义不理解…；"不理解含义""其他"沉底）；逻辑：**上题=中立/负向时显示**。
- **标签池（示例）**：大牌同厂 / 源头工厂·源头好货 / 百元店 / 折扣仓 / 全网金牌卖家 / 品质质检 / 潮流好货 / 京喜好店。
- **正向原因变体**：部分标签（如"潮流好货"）额外设"持正向原因"多选（逻辑：态度=正向时显示）。
- source：s2店铺 Q5–Q16、Q30–Q31（标签态度二段式，多标签折叠）

### 标签语义区分 / 适配度打分（2026-07-08批）

- **【单选】看到"`[标签A]`"和"`[标签B]`"两个标签，您认为它们代表的店铺是同一类吗？** — 是，感觉一回事 / 否，不应是同一类。
- **【多选/打分·需附图】请浏览以下店铺页，为各店铺与"`[目标标签]`"的适配度打 1–5 分。** — 店铺样例 × 适配度评分（配图投放时插入）。
- **【打分矩阵】为"`[某标签]`"与其他标签的匹配度打 1–5 分。** — 标签×标签匹配矩阵（如"京喜好店 × 潮流好货/品质质检/…"）。
- source：s2店铺 Q22/Q25/Q26/Q32

### 子品牌认知漏斗（听说→关系理解→印象·2026-07-08批）

- **【单选】您是否听说过 `[关联子品牌/目标活动]`（如"京喜"）？** — 听说过且买过 / 听说过没买过 / 没听说过（认知漏斗第一层）。
- **【单选】您认为 `[关联子品牌]` 与 `[本平台]` 是什么关系？** — 认为是自营店 / 认为代表第三方卖家 / 不清楚二者关系。
- **【多选】下面哪些词更符合您对 `[关联子品牌]` 的印象？** — 印象词池（接地气/便宜实惠/极致性价比/品类丰富/源头直供/工厂货…含负向词如质量低劣/浮夸虚假，"其他"兜底）。
- source：s2店铺 Q27–Q29（京喜认知漏斗）· s3金券 Q2–Q3（活动知晓：无提示 + 有提示知晓）

### AI 购物功能整体态度与期待（AI 类研究专用·2026-07-08批）

- **【单选】您对在电商 App 中植入 `[AI功能]` 的态度是？** — 非常认可→非常不认可 5 档。
- **【多选】以下哪些形容词更符合您对 `[AI功能]` 的期待？** — 期待词池（聪明的/高效的/靠谱的/懂我的/博学的/有趣的/耐心的/简洁的/客观的/专业的，"其他"兜底）。
- source：s5 AI动线 Q8/Q27

## 使用说明
- 怎么用：`generate-survey` 组装"态度/认知"段时取本文件；态度分型选项按研究假设定制，可作聚类/画像的分型变量；心智五感与满意度-NPS 度量组适合平台/活动品牌心智研究。
- 注意：态度分型题避免抽象词让用户自我归类，用具体描述句代替（questionnaire-design：抽象概念操作化）；NPS/满意度口径统一取 [[standardized-ux-scales]]。
- 自查：过 [[questionnaire-design]]。
