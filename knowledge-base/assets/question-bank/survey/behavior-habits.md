# 问卷题库 · 行为与习惯（behavior-habits）

> **主题定义**：现状行为的量化题——平台使用、频率、客单价、品类偏好、购买节奏。收"平时怎么做"。
> **占位符约定**：`[目标品类]`/`[目标人群]`/`[本平台]`/`[目标权益]`/`[AI功能]`，复用时替换。

## 题组

### 平台/渠道使用

- **【多选】近 N 个月，您在哪些平台下单过 `[目标品类]`？** — 选项为平台清单（美团/饿了么/`[本平台]`/淘宝闪购/抖音/品牌官方/其他）。
- **【单选·引用】其中您用得最多的平台是？** — 引用上题所选。
- **【多选】近 N 个月，您在下列哪些渠道浏览/挑选过 / 购买过 `[目标品类]`？** — 浏览与购买分两题，可对比"逛"与"买"渠道差。
- source：qb5 Q2–Q3 · qb6 渠道选择
- **变体（竞品会员/月卡持有·2026-07-08）**：除 `[本平台目标权益]` 外，您还使用过下列哪些平台的月券包/月会员？ — 竞品会员清单（选项随机，"没使用过"兜底）。
  - source：s4外卖月卡 Q22

### 频率与客单价

- **【单选】近 N 个月，您用 `[目标品类]` 的频次大概是？** — 锚点用客观频次（每周8次+/5–7/3–4/1–2/每月≤3），**不用"经常/偶尔"**。
- **【单选】给 `[目标人群]` 用的频率？**（人群特化）
- **【单选】平均每次花费大概是？** — 金额区间（分档）。
- **【单选】有促销时一次购买金额？** — 与日常价对比。
- source：qb5 Q5–Q6 · qb3 Q7 · qb6 品类购买习惯
- **变体（活动/权益使用频率·2026-07-08）**：近 N 天，您来抢 / 使用 `[目标权益]`（如外卖金券）的频率是？ — 客观频次（几乎每天/每周4-5天/每周2-3天/每周1天/每月2-3天/每月≤1天）。
  - source：s3金券 Q9 · s4外卖月卡 Q4
- **变体（外卖频率·标准分档·2026-07-08）**：最近 N 个月，您在 `[本平台]` 点 `[目标品类]` 的频率是？ — 几乎每天/每周4-5次/每周2-3次/每周1次/每月2-3次/每月≤1次/几乎没有。
  - source：s3 Q23 · s4 Q25

### 品类偏好

- **【多选】您用过 / 买过下列哪些 `[目标品类]` 子类？** — 子类清单（选项随机、"其他"沉底）。
- **【矩阵/多选】`[目标人群]`（自己/孩子/老人）分别爱吃哪些？** — 按成员拆分偏好。
- source：qb3 Q8 · qb6 品类食用习惯
- **变体（外卖品类偏好·2026-07-08）**：您平时点 `[目标品类]`，哪些点得比较多？ — 品类清单（咖啡奶茶/甜点烘焙/中式快餐/西式快餐/特色小吃/其他）。
  - source：s4外卖月卡 Q26

### 购买节奏与包装（品类特化）

- **【单选】通常多久购买一次？** / **一周什么时候买？** / **一次买多少钱（到手价）？** / **偏好什么分量的包装？**
- **【单选】第一次给 `[目标人群]` 用 `[目标品类]` 时，TA 的年龄是？**（起始年龄，人群特化）
- source：qb6 品类购买习惯 · qb3 Q6

### 品类消费行为量化补充（跨品类·2026Q2）

- **【多选·选项随机】近 N 个月，您购买过以下哪些 `[品类大类]` / `[目标品类]` 子类？** — 品类清单勾选（甄别 + 行为双用）。
- **【单选】您单次购买多少 `[目标品类]`？** — 购买量档（试用装 / 1–2 件 / 整箱 / 多箱…）。
- **【单选】您购买 `[目标品类]` 的习惯更接近哪种？** — 品牌忠诚度分型：只买同一款 / 固定几个品牌 / 不看重品牌看度数口感 / 看价格促销 / 爱探索尝新。
- **【单选】您更倾向哪种购买方式？** — 一次买齐 / 分阶段逐步买 / 视情况（适用套装、囤货类）。
  - 注：频次、单次/促销花费已由上方「频率与客单价」覆盖，此处只补购买量 / 忠诚度 / 节奏维度。
- source：电动牙刷 S4/S5 · 光瓶酒 A21/A25 · 奶粉 Q14/品牌偏好 · 厨电 Q8/Q9 · 特价 Q4（2026Q2）

### 会员 / 平台使用行为（2026Q2）

- **【多选】您目前正在开通/使用哪些 `[某类]` 会员/服务？** — 会员清单（电商付费会员 / 金融产品 / 支付方式）。
- **【单选】您对 `[本平台]` 的使用情况是？** — 使用深度分档：知道没用过 / 下载没买过 / 只买大件 / 急需才买 / 买得少 / 经常买爆品 / 各种都买。
- **【多选】您曾为 `[对象]` 在哪些平台开通过 `[档案/会员]`？** + **【多选】开通的主要原因？** — 会员开通行为 + 动机。
- source：PLUS S1/E1/E2 · 特价 Q33 · 电动牙刷 Z1 · 宠物 Q5/Q6（2026Q2）

### 品类使用现状（品类特化·2026Q2）

- **奶粉**：目前喝的品牌 / 第一罐品牌 / 喝奶粉阶段（初尝/转奶/主食/辅食/衰退）/ 了解与准备时点 / 品牌更换契机 / 试用装作用。
- **电动牙刷**：使用情况（在用/换回手动/考虑中/从未）/ 品牌 / 价格段 / 刷头更换频率。
- **白酒**：喝过的酒类型 / 光瓶 vs 盒装占比 / 度数·规格偏好 / 尝试低度前的酒类经历。
- **驱虫**：购买的品类类型 / 遇到的虫害 / 首次或持续 / 购买月份 / 产品形态 / 品牌认知与购买。
- source：奶粉 Q7/Q12/Q13/Q10/Q11/Q22/Q21 · 电动牙刷 S6/C1/C2/B1/B6 · 白酒 A2/A3/A7/A8 · 驱虫 S9/A1/A2/A3/A14–A19（2026Q2）

### 差旅/出行行为（商旅类专用·2026-07-08）

- **【单选】近 N 个月，您的出差频率如何？** — 客观频次锚点（每周2次+/每周1次/每月2-3次/每月1次/近3月1-2次/近3月未出差）。
- **【多选】您出差的主要目的通常是？** — 目的清单（拜访客户/展会/内部会议培训/项目驻场/异地团队管理/其他）。
- **【单选】您出差目的地的描述？** — 城市线级分布（一线新一线省会集中 / 非省会地县级 / 各线级都有 / 境外港澳台〔可设终止〕）。
- **【单选】您出差一次的平均周期是？** — 1天内 / 2-3天 / 4-5天 / 5天以上。
- source：s1商旅 Q1–Q4（2026-07-08）

### AI 购物功能使用现状（AI 类研究专用·2026-07-08）

- **【多选】近期您注意到以下哪些电商 App 植入了 AI 购物功能？** — 平台清单（选项随机，"没注意到"沉底互斥）。
- **【多选】您在电商 App 注意到过 / 使用过哪些类型的 `[AI功能]`？** — 功能类型池（AI搜索/拍照识图/生成购买清单/对比商品/评价总结/参数解读/试穿/凑单/找低价/价格趋势；"使用过"题引用"注意到"题，构成注意到→使用漏斗）。
- source：s5 AI动线 Q4/Q6/Q7（2026-07-08）

## 使用说明
- 怎么用：`generate-survey` 组装"消费行为"段时取本文件；频率/金额锚点按品类调整分档；品类使用现状题按目标品类择用、注明"品类特化"；商旅研究取"差旅/出行行为"、AI 研究取"AI 购物功能使用现状"。
- 注意：频率/程度类**必须给可量化锚点**（questionnaire-design MUST 4）；避免"经常/有时/很少"。
- 自查：过 [[questionnaire-design]]（锚点统一、一题一事、选项穷尽互斥）。
