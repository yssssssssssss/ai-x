# 问卷题库 · 体验与痛点（experience-painpoints）

> **主题定义**：使用/购买中的不便、阻碍、以及对本平台体验的负向评价（量化）。收"哪里不方便/不满意/为何不选"。
> **占位符约定**：`[目标品类]`/`[目标人群]`/`[本平台]`/`[目标权益]`/`[AI功能]`，复用时替换。

## 题组

### 场景痛点

- **【多选·选项随机】在 `[引用场景]` 用 `[目标品类]` 时，下列哪些情况让您觉得不方便或比较麻烦？** — 痛点池（如：没有专区一家家翻费时 / 份量多是成人标准找不到小份 / 难找营养均衡的 / 手动备注麻烦 / 备注了商家不履约 / 其他）。
  - 说明：原问卷对不同场景各设一版同构痛点题（Q14/Q18/Q21），**去重折叠为 1 个题组模式**，`[引用场景]` 随分支替换；各场景独有痛点保留为该场景选项。
- source：qb3 Q14 / Q18 / Q21（三场景折叠）

### 平台体验痛点

- **【多选】针对在 `[本平台]` 购买 `[目标品类]`，您遇到过哪些体验不好的地方？** — 平台侧痛点池。
- **【多选】（变体·月卡/权益整体问题）您觉得 `[目标权益]`（如外卖超级月卡）整体体验上还有哪些问题？** — 权益整体问题池（权益太少 / 没吸引力 / 门槛高 / 怕自动续费 / 怕不能退款 / 有效期短 / 宣传与实际不符；"其他"沉底）。
- source：qb6 渠道选择（京东体验不好的地方）· s4外卖月卡 Q17（月卡/权益整体问题变体，2026Q2）

### 未选本平台的原因

- **【多选/单选】`[本平台]` 不是您最常购买 `[目标品类]` 的平台，主要原因是？** — 流失/未转化归因（仅对"非本平台首选"用户显示）。
- source：qb6 渠道选择（京东非首选原因）

### 使用体验痛点（产品侧·跨品类·2026Q2）

- **【单选】您对目前使用的 `[目标品类]` 满意吗？** — 5级满意度（配合 [[mindset-cognition]] 的满意度-NPS 度量组）。
- **【多选·限N项·选项随机】使用 `[目标品类]` 过程中，您遇到的最大痛点或未达预期的问题是？** — 痛点池（分层：基础体验缺陷 / 核心效果未达预期 / 耗材相关 / 智能功能鸡肋 / 维护不便）。
- **【多选·限N项】促使您 停用/换回/放弃 `[目标品类]` 的主要原因是？** — 流失原因池（产品体验侧：噪音 / 续航 / 效果未达 / 复杂难上手 / 适应期不适 / 导致新问题 / 耗材贵 / 他人建议）；与 [[decision-journey]] 的渠道流失互补。
- source：电动牙刷 C4/C5/C7（2026Q2）

### 平台/权益改进点（2026Q2）

- **【多选·选项随机】您觉得 `[平台/频道]` 在哪些方面还需要改进？** — 负面体验池（价格贵 / 不包邮 / 优惠少 / 优惠难领 / 任务麻烦 / 奖励不实用 / 种类少 / 质量没保障 / 不够有趣），与正面体验题对照。
- **【多选】您近期在 `[平台]` 购买 `[目标品类]` 不满意的原因是？** — 满意度下探痛点池（商品 / 价格 / 物流 / 客服 / 会员权益 多层）。
- **【多选】您认为 `[会员某权益]` 还有哪些需要提升的地方？** — 权益改进池（折扣力度 / 独有性 / 覆盖度 / 品质感 / 易得性 / 易用性 / 体验感，按餐饮/外卖/出行/娱乐/生活/金融各权益），可复用为 `[权益]` 改进诊断。
- source：特价 Q30 · 文具 Q17/B16 · PLUS A11–A16（2026Q2）

### 权益/券使用漏斗 —— 「用没用 → 未用原因 → 满意度 → 使用问题」四联 battery（2026Q2）

> **battery 模式（可复用）**：对每个权益跑一套 4 题——使用情况单选 →（没用）未使用原因多选 →（用了）满意度单选 →（用了）使用问题多选；`[目标权益]` 替换即可复用。s4 对"外卖折扣券 / 1v1 急送体验券 / 自营家政券"各跑一遍，三权益折叠为 1 套模式。

- **同一权益依次问 4 题（后 3 题带显示逻辑）**：
  1. **【单选】您目前使用过 `[目标权益]` 几次 / 是否使用过？** — 次数分档 或 已用/还没用。
  2. **【多选·选项随机】您没使用 `[目标权益]` 的主要原因是？** — 未用原因池（想买的用不了 / 没想起来 / 最近没需求 / 门槛高 / 其他）；逻辑：**上题=没用时显示**。
  3. **【单选】您对 `[目标权益]` 的使用满意吗？** — 非常满意→非常不满意 5 档；逻辑：**用过时显示**。
  4. **【多选·选项随机】您觉得 `[目标权益]` 实际使用中主要有哪些问题？** — 问题池（每单减得少 / 减多少不确定 / 常有不支持 / 数量多用不完 / 提示不明显…）；逻辑：**用过时显示**。
- **权益池（示例·外卖月卡）**：外卖折扣券 / 1v1 急送体验券 / 自营家政券。
- source：s4外卖月卡 Q4–Q15（三权益四联 battery 折叠）· s3金券 Q16–Q17（用券情况 + 未用原因）

### 金券/活动全漏斗痛点（看到没参与 / 参与后不再 / 不满意原因）

- **【多选·选项随机】您看到过 `[目标权益]` 但没参与的主要原因？** — 原因池（没赶上时间 / 门槛高 / 力度不够 / 要记时间 / 时段太多 / 规则不清 / 有效期短 / 中奖预期低 / 外卖少 / 怕套路 / 不感兴趣；"其他"沉底）；逻辑：`来看过但没抢`时显示。
- **【多选·选项随机】您之前抢过但现在不参与的主要原因？** — 同类原因池；逻辑：`之前抢现在不抢`时显示。
- **【多选】您对 `[目标权益]` 活动不满意的主要原因？** — 逻辑：满意度=一般/不满意时显示。
- **【文本】参与 `[目标权益]` 时遇到过哪些问题？有哪些改进建议？** — 开放题（各原因池已备注"优惠力度/门槛/玩法/适用范围/有效期/入口/中奖预期"等归因标签，便于编码）。
- source：s3金券 Q7 / Q15 / Q19 / Q20

### AI 功能"用处不大原因" —— 分功能 battery（2026Q2）

> **battery 模式（可复用）**：对每种被使用的 AI 功能追问"觉得用处不大的原因"，`[AI功能]` 替换即可复用。s5 对"AI搜索 / 拍照识图 / 生成清单 / 对比商品 / 评价总结 / 参数解读 / 试穿 / 凑单 / 找低价"各跑一遍，9 功能折叠为 1 套模式。

- **主问（多选·每功能一题，带显示逻辑）**：**您觉得 `[AI功能]` 用处不大的原因是？** — 该功能专属原因池 + "其他"沉底；逻辑：**用过该功能 且 作用评分≤7 时显示**，选项随机。
- **原因池（示例·各功能）**：AI搜索→不会描述需求 / 结果太多没耐心看 / 没按偏好推 / 答非所问 / 太慢 / 排序不懂 / 和普通搜差不多；评价总结→只有好评没差评 / 无追评 / 无法辨刷评 / 太笼统 / 不能按维度筛 / 无图 / 无提及占比 / 营销感重；AI试穿→效果不像 / 不还原身材 / 怕美化 / 支持商品少 / 卡顿 / 不能多角度 / 不能试整套 / 隐私顾虑；找低价→店铺不靠谱 / 只看价不看质 / 不说差价原因 / 非实时价 / 无相似替代 / 不能组合找低价…
- source：s5 AI动线 Q13–Q21（9 功能"用处不大原因" battery 折叠）

### AI 使用总痛点 / 放弃原因 / 改进方向（2026Q2）

- **【多选】使用 AI 购物功能过程中，您遇到过哪些问题？** — 通用问题池（结果不准 / 来源不透明 / 操作复杂 / 信息不全 / 信息太复杂 / 与传统割裂 / 像打广告 / 入口难找 / 隐私顾虑；"没遇到"沉底）。
- **【多选】您不使用 AI 购物功能的原因是？** — 逻辑：`日常不使用`时显示（不清楚能做什么 / 来源不透明 / 操作麻烦 / 割裂 / 入口难找 / 不信任 / 隐私 / 没植入我要的功能）。
- **【多选】您希望 AI 购物功能在哪些方面改进？** — 改进方向池（更准更智能 / 信息更全 / 更简单流畅 / 更客观可信 / 更懂我 / 更主动互动 / 覆盖更多场景 / 更高效 / 更有趣 / 更集中好找；"没期望"沉底）。
- source：s5 AI动线 Q24 / Q25 / Q26

## 使用说明
- 怎么用：`generate-survey` 组装"痛点诊断"段时取本文件；痛点池按品类替换，可作为改进优先级（配 [[issue-prioritization]] / IPA）的输入；满意度题口径取 [[standardized-ux-scales]]。
- 注意：痛点题选项力求"具体可归因"（对齐访谈侧 [[experience-issue-description-script]] 的具体化要求）；"未选原因"题设显示逻辑。
- 自查：过 [[questionnaire-design]]（选项穷尽互斥、非诱导）。
