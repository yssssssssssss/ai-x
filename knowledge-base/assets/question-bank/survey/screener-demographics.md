# 问卷题库 · 筛选与背景（screener-demographics）

> **主题定义**：甄别题（含终止逻辑）与人口统计题。收"是否合格 + 是谁"。
> 与相邻主题边界：现状行为归 [[behavior-habits]]（survey）。
> **占位符约定**：`[目标品类]`/`[目标人群]`/`[本平台]`/`[目标功能]`/`[目标活动]`，复用时替换。问卷题保留 **题型 / 选项要点 / 逻辑**。

## 题组

### 甄别题（含终止逻辑）

- **【单选】近 N 个月，您是否下单过 `[目标品类]`（不限平台）？** — 选项：是 / 否 / 记不清了；逻辑：≠"是" 终止。
- **【单选】您是否为 `[目标人群]` 相关身份？**（如"为人父母，家中有 12 岁及以下子女"）— 选项：有 / 没有；逻辑：否 终止。
- **【单选】给 `[目标人群]`（或与其一起）用 `[目标品类]` 时，您的参与程度是？** — 独自决定 / 我为主但征求意见 / 共同商议 / TA 说了算我只买单；逻辑：仅决策参与者续答。
- **【多选·甄别】近 N 个月，您在线上买过下列哪些 `[品类大类]` / 哪些 `[目标品类]`？** — 用于筛"品类用户"，选项为品类清单。
- source：qb3 亲子餐 Q1–Q3 · qb5 外卖信息 Q1 · qb6 速冻面点 用户甄别

### 行为甄别（活动/功能类·含终止·2026-07-08）

- **【单选·甄别】近 N 天，您是否在 `[本平台]` 浏览过 / 使用过 `[目标品类/目标功能]`？** — 浏览过且下过单 / 浏览过但没下单 / 没进入过 / 记不清了；逻辑：后两项**提前结束**。
- **【多选·甄别】近 N 个月，您在以下哪些 App 逛购过 `[目标品类]`？** — 平台清单（选项随机，"都没用过"沉底互斥 → **结束作答**）。
- **【单选·渠道甄别】您平时习惯通过什么渠道购买 `[某品类]`？** — 线上 / 线下 / 很少购买（"很少购买"用于后续显示逻辑分流）。
- source：s3金券 Q1 · s5 AI动线 Q1 · s2店铺 Q2/Q3（2026-07-08）

### AI/新技术态度甄别（AI 类研究前置·2026-07-08）

- **【单选】对于当前涌现的各类 AI 新技术，您的态度更倾向于？** — 积极开放愿尝试 / 中立观望 / 保守有顾虑 / 完全不关注。
- **【多选】您日常常用的 AI 工具有哪些？** — 工具清单（豆包/DeepSeek/元宝/通义/Kimi… 选项随机，"日常不使用"沉底互斥）。
- source：s5 AI动线 Q2/Q3（2026-07-08）

### 多级消费甄别漏斗（可复用范式·2026Q2）

> 用"大类消费 → 品类细分 → 是否在用/流失/潜在 → 谁买谁用"多级漏斗精准筛人，可复用为 `[目标品类]` 的甄别链路。

- **【多选·选项随机】近 N 个月，您进行过/购买过以下哪些 `[品类大类]` 消费？** — 大类清单；逻辑：未选中目标类 → 终止。
- **【多选】近 N 个月，您具体进行过哪些 `[目标品类]` 细分消费？** — 细分清单；逻辑：未选中核心项 → 终止（二级甄别）。
- **【单选】您目前使用 `[目标品类]` 的情况是？** — 在用 / 曾用现已流失 / 从未用但考虑中 / 从未用也不考虑；逻辑：分流 + 终止，用于区分 已购/流失/潜在 样本。
- **【单选】您使用/曾用的 `[目标品类]` 由谁购买？** — 我自己 / 他人（他人常终止，保证受访者是决策者）。
- **【矩阵多选】您购买的 `[各品类]` 主要是谁使用？** — 品类×使用者矩阵（自用甄别）。
- **【单选】针对您考虑购买的 `[目标品类]`，哪种描述更符合您？** — 潜在用户细分（已做功课有明确目标 / 对比中纠结 / 大概了解 / 还没做功课，末项常终止）。
- source：电动牙刷 S4/S5/S6/S7/S8 · 驱虫 S9 · 文具 S4（2026Q2）

### 品类专属甄别（品类特化·2026Q2）

- **奶粉**：婚育情况（已婚有娃续答）· 宝宝年龄（含终止）· 是否喝奶粉 · 购买参与度 — 母婴多级甄别链。
- **白酒**：近 6 月是否饮酒（无则跳转）· 饮用过的酒类型 · 光瓶/盒装占比 — 饮酒甄别。
- **驱虫**：购买的驱虫品类 · 是否有宠物 · 户外驱蚊场景甄别（仅户外驱蚊用户续答）· 住宅环境 — 场景化甄别。
- **特价频道**：是否浏览/购买过频道（没见过终止）· 打开频次 — 频道用户甄别。
- **生鲜流失**：过去买过但近期未再买的流失定义（题干内置甄别）— 流失用户甄别。
- **文具**：是否有孩子（终止）· 孩子教育阶段（仅小学初中续答）· 年级 · 性别 — 家长 + 学龄甄别。
- source：奶粉 Q3–Q6 · 白酒 A1–A3 · 驱虫 S9/S8-1/S11/S12 · 特价 Q1/Q2 · 生鲜 A1 · 文具 S3/S5/S6/S8（2026Q2）

### 招募甄别（招募期·2026Q2）

> 招募问卷的甄别题可复用；**PII 采集题（姓名 / 联系方式）与访谈排期题不入库**（含个人信息，按 sensitivity-rules 剔除，仅在转换计划留痕）。

- **【多选】您家目前有哪些宠物？** — 猫 / 狗 / 小宠 / 水族 / 没有宠物（排他，养宠前提甄别）。
- **【单选】您养宠物多长时间了？** — 时长甄别（不到半年 / 半年及以上）。
- **【多选】近 3 个月，您为宠物购买过哪些用品/服务？** — 宠物消费清单（主粮/零食/保健/药品/洗护/清洁/玩具/窝垫出行/智能用品/服饰/美容/医疗/问诊/寄养/训练）。
- ⚠️ **不入库**：宠物招募的访谈排期题与个人信息采集题（姓名 + 联系方式）——招募流程题 + PII，按规则剔除，仅在转换计划留痕。
- source：宠物招募 Q1/Q2/Q3（Q10–Q12 已按 PII 规则剔除）（2026Q2）

### 人口信息（问卷尾部）

- **【单选】性别** / **年龄**（分段）/ **婚育情况** / **`[目标人群]`年龄** / **居住情况** / **城市线级** / **家庭月收入**（区间）。
  - 注意：收入等敏感题设"不便透露"兜底、不强制应答（见 questionnaire-design）。
- source：qb3 Q35 · qb6 人口信息 · qb5 甄别配额
- **分档变体池（2026Q2·按目标人群择用，同一维度多种分档口径备选）**：
  - 年龄：18-25/26-30/31-35…（消费类细档）· 18岁以下/18-25/26-35/36-45/46-55/56+（宽档）· 含终止的年龄甄别档。
  - 婚育：未婚 / 未婚恋爱中 / 已婚无小孩 / 已婚有小孩(含孕) / 其他；或 未婚/已婚无娃/已婚有娃/其他。
  - 居住：独居 / 与朋友室友合住 / 与伴侣住 / 与伴侣孩子住 / 与孩子住 / 与父母长辈住 / 其他。
  - 城市：常住城市 +【暗题】城市线级（一二线/三四线/五六线，或按第一财经分级）+ **南北方暗题** + **七大区域暗题**（东北/华北/西北/华东/华中/华南/西南）。
  - 学历：初中及以下 / 高中中专 / 大专 / 本科 / 硕士及以上（含博士档）。
  - 职业：政府事业 / 国企 / 教师 / 自由职业 / 个体 / 专业技术 / 企业主管 / 职员 / 务工 / 学生 / 无业 / 退休 / 主妇 等。
  - 收入：家庭税后月收入 / 个人月可支配 / 家庭可支配月收入（"未婚填个人、学生填生活费"说明）；分档从"4000及以下…30000+"到细分档。敏感题设兜底、不强制。
  - 家庭教育/决策角色：孩子爸爸 / 妈妈 / 两者一起（母婴、家长类特有）。
  - source：奶粉 Q2/Q3/Q49/Q49b/Q50 · 白酒 · 特价 Q35–Q37 · 生鲜 · 宠物 Q8/Q9 · 电动牙刷 S2/S3/Z3 · 文具 S2/Q77/Q78/Q79 · 驱虫 S3–S6（2026Q2）
- **变体（问卷尾部标准人口模块·s1-s5 五份同构·2026-07-08）**：性别 / 年龄（分段）/ 婚育状态 / 学历 / 职业（系统选项池）/ 家庭月收入（区间，末档"保密/不方便透露"）/ 常驻省市（城市控件，后台匹配线级）——s1–s5 五份问卷完全同构，是稳定可复用的问卷尾部模块；收入题必设兜底、城市题用级联控件。
  - source：s1 Q28–Q29 · s2 Q33–Q39 · s3 Q24–Q30 · s4 Q28–Q33 · s5 Q29–Q35（2026-07-08）

## 使用说明
- 怎么用：`generate-survey` 组装甄别段（问卷开头）与人口信息段（问卷结尾）时取本文件；终止逻辑按目标人群定义调整；多级消费甄别漏斗按"大类→细分→在用/流失/潜在→谁买谁用"裁剪；活动/功能类研究取"行为甄别"、AI 研究前置取"AI 态度甄别"。
- 注意：甄别题放最前、带跳转/终止；人口题放最后、避免开头就问敏感信息；**招募类 PII 与排期题不入库**。
- 自查：过 [[questionnaire-design]] 检查清单（甄别逻辑、敏感题兜底、选项穷尽互斥）。
