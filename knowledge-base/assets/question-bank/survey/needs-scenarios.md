# 问卷题库 · 需求与场景（needs-scenarios）

> **主题定义**：场景枚举、场景触发原因、场景占比的量化题。收"什么情况下用/买"。
> **占位符约定**：`[目标品类]`/`[目标人群]`/`[场景]`，复用时替换。

## 题组

### 下单/使用场景枚举

- **【多选·限N项】近 N 个月，您用 `[目标品类]` 的主要场景有哪些？** — 场景清单（早餐/工作日正餐/周末正餐/下午茶/夜宵/聚会/旅游出差/极端天气/其他），限选前 N 项以逼出优先级。
- **【多选】您平时点 `[目标品类]`，哪些情况下点得比较多？** — 场景清单（早餐/工作日正餐/周末正餐/下午茶/深夜夜宵/聚会聚餐/异地旅游/异地出差/恶劣天气/其他；"以上均没有"兜底）。
  - 说明：外卖场景清单变体，与存量场景枚举（qb5 Q4）高度同构，作为外卖 domain 的场景选项池变体并入，不新建题组。
- source：qb5 Q4 · qb6 品类购买习惯（什么情况下购买） · s4外卖月卡 Q27（外卖场景清单变体）

### 人群/对象场景

- **【多选】给 `[目标人群]` 用 `[目标品类]` 时，有下列哪些情况？** — 如"单独给 `[目标人群]` / 全家一起 / 聚会一起"，用于三大场景覆盖。
- **【单选】在 `[引用上题场景]` 下，一次花费大概多少？**（场景×客单价交叉）
- source：qb3 Q11–Q12

### 细分场景触发原因

- **【多选·选项随机】回想 `[引用场景]` 时，通常出于什么原因？** — 原因清单（如：加餐 / 生病安抚 / 表现奖励 / 补课吃饭 / 垫肚子 / 生日节日 / 家里不便做 / 换口味解馋 / 其他沉底）。
  - 说明：原提纲对不同场景（一人食/全家/聚会）各设一版同构题（Q13/Q17/Q20），**去重折叠为 1 个题组模式**，`[引用场景]` 随分支替换。
- source：qb3 Q13 / Q17 / Q20（三场景折叠）

### 购买触发场景（六分法·可复用范式·2026Q2）

- **【多选·选项随机】近 N 个月，您在哪种情况下购买过 `[目标品类]`？** — 触发场景池：应急购买（已发现问题着急用）/ 提前备货（固定时间备着）/ 种草购买（被直播·博主种草）/ 凑单购买 / 顺手购买（平台推荐）/ 补货购买（用完补）。
- **【单选】在 `[应急场景]` 下，您同时买了几种 `[目标品类]` 来解决问题？** — 应急场景下的购买件数。
- **【多选】您购买/使用 `[目标品类]` 的目的是？** — 目的池（如饮酒目的：缓解情绪 / 自由轻松 / 拉近感情 / 商务气氛 / 日常习惯 / 随餐饮品 / 品鉴 / 不得不喝）。
- **【单选】您使用 `[目标品类]` 的 `[住宅/环境]` 属于哪种？** — 环境分型（如住宅：自建房 / 老小区 / 新小区高低层，影响需求强度与形态）。
- source：驱虫 A3-1/A3-2/S12 · 光瓶酒 A22 · 低度酒 A3 · 白酒购买场景（2026Q2）

### 场景枚举补充（品类场景池 + 子类×场景矩阵·2026Q2）

- **【多选·选项随机】近 N 个月，您在哪些场景下 使用/购买/饮用 `[目标品类]`？** — 品类场景池样例：
  - 饮酒场景：朋友聚会 / 商务应酬 / 一人独酌 / 家庭随餐 / 大型家庭聚餐 / 婚宴寿宴 / 情侣二人 / 露营旅游户外 / 送礼 / 收藏。
  - 驱虫场景：家中室内 / 家中室外 / 户外活动 / 商铺办公。
  - 通用可复用为 `[场景池]`（与「下单/使用场景枚举」互补：此处偏品类特化的丰富场景池）。
- **【多选/矩阵】不同 `[子类]` 分别更适合哪些场景？** — 子类×场景矩阵（如高度酒 vs 中低度酒适配场景）。
- source：光瓶酒 A5a/A5b · 低度酒 A5/A6/A13 · 驱虫 S10/S11（2026Q2）

## 使用说明
- 怎么用：`generate-survey` 组装"需求场景"段时取本文件；场景清单与原因清单按品类替换，配额/交叉分析可用场景做分层变量。
- 注意：场景/原因清单力求穷尽互斥、"其他"沉底兜底；限选项数以产出优先级。
- 自查：过 [[questionnaire-design]]。
