# skills/ · 技能库（可执行生成器）

> 吃一个研究 brief、直接产出交付物的技能。每个技能一个文件夹，含 `SKILL.md`（触发描述 + 生成步骤 + 调用哪些 method/asset/model）+ 支撑 `references/`（部分含 `scripts/`）。
> **关键**：技能不重造内容，而是**编排其余各层的正典**——SKILL.md 通过路径/标签 cross-walk 到对的方法、量表、题库、模型，保证产出与单一出处一致。这是面向上千消费者最高杠杆的一层（建一次，全员用无数次）。
> **边界**：`assets/` 是静态零件；skills 是动态组装零件的执行器。

## 技能清单（按产品阶段分组）

> 划分对齐「产品阶段 × 能力」矩阵：**通用·全流程 / 产品规划 / 产品设计 / 产品迭代** 四阶段共 **18** 个面向消费者的生成能力。首列＝中文能力名 + 技能目录。
> 每个技能以「安装包形态」收录：`SKILL.md` + `references/`（+ 需要时 `scripts/`），不含 evals。
