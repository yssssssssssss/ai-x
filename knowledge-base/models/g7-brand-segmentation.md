# G7 品牌人群细分模型

> 来源：神灯圈子·翁鸿诚《品牌研究的人群细分模型-G7模型》 原文 http://xingyun.jd.com/shendeng/article/detail/46632
> 本文原作者翁鸿诚，写于 2024.05.10。

> 一句话：用"知晓 / 购买 / 考虑 / 首选"四道关把消费者切成 7 组，再按各组比例算出品牌态度指数（A 值），看一个品牌在品类里到底有多"强"。
> 落点：`models/`。判据——拆开就讲不通、贯穿全程才成立（G7 贯穿问卷设计→人群划分→态度指数→策略归因）。
> 边界：models 描述「被研究的东西本身长什么样 / 好不好」；「研究怎么做对」属 `standards/`；「怎么算/出图」属 `toolbox/analysis/`。

## 核心概念

G7 模型是品牌评估的一种**定量模型**：基于消费者对目标品牌的使用情况和消费态度，将消费者分成 7 组（G1–G7），再根据细分人群的比例计算**品牌态度指数**（也被称为"A 值"模型，A-Attitude），用以了解消费者对品牌的喜好程度，进而判断本品牌在行业中的地位、与其他竞对品牌的差距。

它把"品牌有多强"这件事拆成一条可量化的链路：

- 用四道问卷题（知晓→购买→未来考虑→首选）把人群切成 7 类；
- 统计每个品牌 7 类人群的占比；
- 按固定权重系数加权得到该品牌的 A 值（品牌态度指数），横向比竞品、纵向比历史。

**G7 模型的优点：**

1. 操作简单，不需要使用聚类分析等比较复杂的分析方法；
2. 人群划分明确且落地性强，有明确规范的划分标准，可得到不同细分人群的品牌态度及存在的问题，进而寻找对应的策略。

适用场景：品类内品牌的"初步扫描"、周期性品牌监测、判断品牌所处市场阶段。输入是近期购买过该品类的随机用户样本与四道核心题；输出是各品牌的 7 类人群占比、品牌态度指数（A 值）、以及各群对应的品牌问题与策略线索。

## 关键构念 / 维度

G7 的两个核心构念是**人群分群规则**与**品牌态度指数（A 值）**。

### 一、分群依据：四道核心题

人群细分基于对目标品牌的四个二元判断（最基础要求是能区分出这四点）：

1. **用户是否知道该品牌？**（品牌知晓）
   *对应题目：【多选题】提到 xxx 类商品（例如：手机、笔记本电脑等研究目标），您能想到哪些品牌？请选出您想到的全部品牌。*
2. **是否购买过该品牌？**（品牌购买）
   *对应题目：【多选题】过去六个月内，您购买过哪些品牌？（选项引用题目 1 用户知道的全部品牌）*
3. **未来是否会考虑购买该品牌？**（未来考虑购买）
   *对应题目：【多选题】在将来购买该类商品时，您会考虑购买哪些品牌？（选项引用题目 1 用户知道的全部品牌）*
4. **购买时该品牌是否为首选？**（品牌首选）
   *对应题目：【单选题】在将来购买该类商品时，您会首选选择哪个品牌？（选项引用题目 3 用户考虑购买的全部品牌）*

### 二、7 类人群（G1–G7）

选取一个目标品牌，依据"品牌知晓 / 品牌购买 / 未来考虑购买 / 品牌首选"四道题的组合，将用户划分为 7 类。完整的划分对照见下图：

![](https://img14.360buyimg.com/ling/jfs/t1/247260/13/4664/51102/65d317b3F102ac17e/7e5327a12ba03439.png)

完成细分后，可根据每一类用户的分类特点追加针对性问题，以便分析品牌传播、产品策划、销售策略等环节存在的问题并寻找解决方法。各群的营销含义与可追加问题见下图：

![](https://img30.360buyimg.com/ling/jfs/t1/163841/2/43018/55449/65d317e7F3c401f54/9c8a3aca00e2652b.png)

> 原文中各群（G1–G7）的精确判定组合与逐群营销含义以上述两张图为准；正文仅在"应用"部分点名了部分群体的特征（见下）：
>
> - **G1**：未认知本品牌的用户（不认识、不熟悉该品牌）。其比例偏高常意味着品牌仍处导入期。
> - **G2**：认知、但没买过、未来也不愿意购买。其比例偏高说明用户对品牌存在误解，或了解程度较低且缺少了解动力。
> - **G2、G3、G4**：原文将其作为"尝试受阻"的重点人群——这些群体下降时，需排查阻碍尝试的原因（价格过高 / 促销规则 / 品牌形象与用户理想存在差异等）。

## 如何作为透镜使用

G7 属于【品牌评估】类透镜，回答"本品牌在品类里有多强、卡在哪一环、处于什么市场阶段"。它贯穿从问卷设计到策略归因的全过程。

**问题定义阶段**

1. **选目标品牌与品类**：明确要评估的本品牌及对标竞品集合。
2. **设计四道核心题**：按上文四道题搭好问卷骨架；可按需补充无提示品牌提及、品牌优劣势、触媒习惯等题（见下"采集模板"）。

**采集阶段**

3. **抽样投放**：为保证样本信度，随机选择近 6 个月内购买过该品类的用户，不倾向具体某一品牌（避免样本过于集中），并避免购买周期过长导致的遗忘 / 混淆。

**分析 / 结论阶段**

4. **人群划分**：按四道题把用户切成 G1–G7。
5. **算 A 值**：统计各品牌 7 类人群占比，用固定权重系数算出品牌态度指数，横向比竞品、纵向比历史。
6. **归因与回流**：对暴露出的问题溯源——通过问卷其他题或针对性用户访谈挖掘更深层原因，再落到品牌传播 / 产品策划 / 销售策略的具体动作上。

### 三类应用价值

- **行业内品牌初步扫描**：算出本品与竞品的态度指数，快速掌握该品类各品牌情况。例如京东买手、类目运营在接手新类目时做一次"大扫描"，从用户视角了解各品牌。
- **周期性监测，明确阶段问题**：周期性监测各品牌，结合各品牌在对应周期内的动作，为后续品牌策略提供输入。监测中——
  - 若 **G2、G3、G4** 下降，重点排查阻碍这些用户尝试的点（价格过高？促销规则有问题？品牌形象与用户理想存在差异？）；
  - 若 **G1** 比例下降，则考虑品牌传播选择的渠道、形式是否有问题，是否与未认知本品牌用户的触媒习惯存在较大差异。
- **了解品牌所处的市场阶段**：按人群比例判断品牌处于开发期 / 导入期 / 成长期 / 成熟期 / 衰退期中的哪个阶段。
  - G1 人群较高 → 多数用户不认识、不熟悉该品牌，很可能仍处**导入期**；
  - G2（认知、没买过、未来也不愿买）比例高 → 用户对品牌存在误解或了解程度低且缺少了解动力，处于**导入期与成长期之间**，需加强与用户连结并进一步了解需求。

![](https://img12.360buyimg.com/ling/jfs/t1/157131/14/22452/33659/65d3196dFd1ccc98a/49fafbc484f725d4.png)

### 采集模板：可选补充题

四道核心题之外，可按需补充：

- **A. 无提示品牌提及**：*【主观题】提到 xxx 类商品（例如：手机、笔记本电脑等研究目标），您会首先想到哪个品牌？您还能想到哪些品牌？（写出您想到的全部品牌）*
- **B. 品牌优劣势**：*【多选题】请问您为什么选择购买 xxx 品牌？请问您为什么将 xxx 品牌作为首选？*
- **C. 触媒习惯**：*【多选题】请问您曾经看到过该类型商品中，以下哪些品牌的广告？* / *【多选题】请问您曾经在哪些渠道看到过这些广告？*

## 与方法 / 分析的衔接

概念落到可操作的采集与分析时看：

- **量表编制**（`methods/toolbox/collection/scale-development.md`）：四道核心题与补充态度题的题项设计、信效度，是 G7 划分可靠的前提。
- **关键驱动因素分析**（`methods/toolbox/analysis/key-driver-analysis.md`）：G7 暴露的"尝试受阻 / 误解"等问题，可结合品牌优劣势、属性满意度数据做驱动因子归因，定位策略抓手。
- **样本量不足时的合群**：单次调研品牌多、每个品牌又分 7 类，部分群体人数可能偏少；可按研究需要将同一品牌的部分人群合并（如把 G2、G3、G4 合并），重点研究该类人群为何不愿尝试本品牌。

## 局限与误用

- **仅基于品牌维度对用户细分**：与品类研究不同，G7 只从品牌维度划分，未考虑品类研究中的用户需求场景、购买习惯、关注因素，也未涵盖用户画像中的性别、年龄、城市线级、价值观等。
- **态度指数未考虑行业差异**：G1–G7 的权重系数为预先设定、不可自行调整；但各行业各品类可能存在差异，因此需横向与竞争对手比较、或纵向与自身历史数据对比，而非看绝对值。
- **人群细分对样本量要求高**：单次调研品牌多、每品牌再分 7 类，易出现部分品牌部分群体人数过少，影响分析结果；可按需合并人群（如 G2+G3+G4）。
- **不溯源则难落地**：模型算完后仍需对数据暴露的问题溯源——通过问卷其他题或针对性用户访谈挖掘更深层原因，否则很难得出有针对性的建议。
