# 用户养成运营模型

> 来源：神灯圈子·邱伟雄《基于"用户养成"运营模型实现用户增长》原文 http://xingyun.jd.com/shendeng/article/detail/46574
> （本文原作者：电灯大师，写于 2020.12.21）

> 一句话：以"用户视角"为起点，把用户从初次接触到主动分享拆成一条养成链路，用来串联运营动作、指导用户增长。
> 落点：`models/`。判据——拆开就讲不通、贯穿全程：从接触→认知→关注→体验→使用→付费→习惯→分享的链路贯穿整个增长运营与拉新留存研究。
> 边界：models 描述「被研究的东西本身长什么样 / 好不好」；「研究怎么做对」属 `standards/`；「怎么算/出图」属 `toolbox/analysis/`。

## 核心概念

- 一句话讲清：源出《进化式运营》的"用户养成运营模型"主张——**"用户视角"才是运营思考的起点，是运营的底层框架**。通过用户视角才能把运营模块有序串联，从而真正盘活整个企业的运营资源。
- 运营的本质：以特定产品（或个人/品牌）为运营对象，通过对运营资源（人力、财物、渠道）的最优化配置，**持续提升目标人群（受众用户）与产品的关联度**（即提升用户黏性，就是运营的目标）。运营有 3 个要素：**特定产品、运营资源、目标人群**。
- 与企业视角模型的对比：网络上常见的增长模型（行为事件分析、用户行为路径分析、点击分析、用户健康度分析、用户画像分析、漏斗模型分析）中，漏斗模型的经典即 **AARRR 模型**（源自《增长黑客》）。AARRR 的特点是**企业视角**——很多公司增长部门用它指导并考核运营。
  - 企业视角的问题：站在企业方立场清楚产品价值（能用、好用、易用），但从用户视角看，用户未必认知产品价值，容易"注册即流失"；注册后不付费、付费中混入大量薅羊毛用户，优质用户在转化中流失，剩下的只是漂亮的新客增长率。
  - 因此需要回归用户视角，让更多用户更持续地为产品买单。

## 理论来源

- 提出者 / 出处：模型出自《进化式运营》一书的"用户养成运营模型"；对照的企业视角 AARRR 模型出自《增长黑客》。
- 本文应用语境：京东用户体验设计部新用户拉新与留存专项研究，作者结合该模型解答"如何在用户视角下通过运营手段实现用户增长"。

## 关键构念 / 维度

把用户养成过程拆解为 **7 大转变过程**（链路上的节点）：初次接触产品 → 初步认知产品价值 → 关注产品（全面了解）→ 初次体验产品 → 决定使用产品 → 为产品付费 → 习惯使用产品 → 心甘情愿地向其他用户分享产品。

**简化模型：用户个体与产品的线性关系**

![](https://img30.360buyimg.com/ling/jfs/t1/132126/33/19749/10382/5fd734c7Ecf0a34a9/68b3e85b7ff50bb8.png)

- **线性 vs 网状**：以用户个体视角，养成近似一个线性过程，这是为了方便理解的**简化模型**；实际"用户养成"过程不会纯粹线性。
- **群体网状结构**：要真正指导现实运营，须回归用户"群体"视角——它是一个充满"联动关系"的网状结构。"圆圈面积"代表用户数量，从底部的接触用户数到顶部的习惯用户数面积**逐渐变小**（同一批首次接触的用户群，随养成阶段上升数量呈递减趋势）。
- **分享与放弃可发生在任意环节**：在养成的任一环节，都可能因运营激励或用户自身原因做出"分享"，也可能因各种原因做出"放弃"（例：用户因想塑造公益、健康形象而分享微信运动；新用户因京东商品价格高于竞品而放弃）。

**群体"用户养成"模型简化总图**

![](https://img10.360buyimg.com/ling/jfs/t1/140738/10/18451/264377/5fd7337dEe45f442b/0578ee761a5244c5.png)

> 并非所有产品的新用户都需完整经历整个过程：对"使用门槛"较低的产品（用户无须付出多少金钱、学习成本即可体验，如休闲娱乐类或业界成熟产品），认知、关注的过程会在很大程度上缩短。

## 如何作为透镜使用

用这条养成链路，把每个阶段对应的研究任务与运营动作串起来：

- **问题定义阶段**：确立"用户视角是起点"，先判断目标用户当前处于养成链路的哪一环、卡在哪个转化节点（接触→认知、关注→体验、体验→使用、使用→付费、付费→习惯、习惯→分享），再定研究与运营目标，避免只盯企业视角的新客增长率。
- **采集阶段（各节点对应的运营/研究任务）**：
  - **接触 → 认知**：① 做深度用户调研，进行**用户认知水平研究**，分析目标群体对产品/服务的认知程度。注意：认知程度 ≠ 知晓程度，新用户对京东的认知远不及固有印象（如低线城市 46 岁以上"网络难民"——不会绑卡、不懂填地址、担心付费不发货/丢件，缺少认知教育，即使开始体验也难完成后续转化）。② 在调研基础上制定接触策略：甄别高密度用户渠道（最高性价比地找到目标用户，如面向好学人群可选知乎、得到）；结合接触目标与渠道确定展示形式；保证"接触内容"包含三要素——**吸引用户注意力、灌输产品价值、推荐关注渠道**。注意不要在接触内容中夹带太多营销，尤其"高门槛"产品，认知教育目的是初步建立信任，推销性内容会失去"中立感"、引发质疑甚至负向口碑。

**用户认知水平研究**

![](https://img14.360buyimg.com/ling/jfs/t1/137982/1/18627/64036/5fd73337Efd94db14/a03e397ec4ad6478.png)

  - **认知 → 关注 → 体验**：对有认知/信任门槛的产品，潜在用户哪怕有兴趣也会顾忌，中间有"关注"过程。优先选最契合接触用户、最符合主流行为习惯的关注渠道，降低关注门槛（注意 00 后用 QQ 多、80/90 用微信多，需精细化渠道运营）。"关注"只说明用户有点兴趣、认知尚浅，应尽量**缩短关注期**，持续打造"信任链"、传递产品价值（如官方抖音号持续发优质内容、与用户互动塑造品牌认知）。
  - **体验 → 使用**：用户进入体验只代表"认知层面"认可价值。关注：产品好不好看？是否好用、易用，功能引导/操作是否被理解、是否符合操作习惯，会不会用起来别扭？是否解决了用户问题、能否找到想要的商品？咨询客服能否有效解决？——这些是用户决定是否在"手机"里留下一席之地、从体验转向使用的关键。
  - **使用 → 付费**：降价、促销只是工具层面措施，时间长了用户摸清套路、不做活动就不转化；"先涨价再搞活动"会让用户觉得被欺骗。要可持续盈利，先回答几个问题再谈营销：用户为什么选我们而非替代品？购买时是否与其他渠道对比、为何选了竞品？我们的独特优势是什么、是否与用户心智契合？为什么是这个价格、价格超预期时用户会怎么做、价差到多少会转向友商？（案例：京东老用户认知里"品质保障、售后好"是优势，但部分新用户认为淘宝官方旗舰店、拼多多百亿补贴同样有保障；一名大学生新用户因 6000 多元电脑比友商贵 300 元而流失。）
  - **付费 → 习惯**：养成中**最耗资、费时最大、也最具商业价值**的环节。现金补贴（如美团、滴滴类 O2O）刺激大、传播快，但过度物质刺激投入产出比不高，甚至养成"送钱就用、不送钱不用"的反向习惯。需要更高性价比或"润物细无声"的方式（如针对价格敏感用户给小恩小惠——淘宝淘金币，每天登录签到领取，虚拟币可抵扣/抽奖，对省钱型用户形成较强黏性）。
  - **习惯 → 分享**：常见手段是分享有礼/双边有礼（多为优惠券、虚拟币等物质激励），初期见效快但易"刺激疲劳"。从用户视角，"分享"本身是用户的强烈心理需求，应思考的不是"如何刺激分享"，而是"如何为用户社交分享的心理需求提供人性化出口"（社交心理需求含：身份标签/彰显与众不同、塑造良好社会形象、刷存在感、吐槽等）。例：微信运动"公益捐步数"让用户既彰显公益心又侧面体现爱运动的阳光形象；直播带货中也可让普通用户做"好物推荐官"，达到社交目的并彰显眼光、品味。
- **分析 / 结论阶段**：以群体网状视角评估各阶段的用户数量递减与流失/分享分布，定位转化漏斗的薄弱环节，输出"符合用户需求、切实可行"的运营策略，目标是培育健康的用户转化漏斗、实现持续有效增长。

## 局限与误用

- **企业视角误用**：仅用 AARRR 等企业视角指标考核运营，易忽视用户认知与价值感知，导致"注册即流失"、新客增长率虚高而优质用户流失。
- **线性误读**：把简化的个体线性链路当成真实路径；实际养成是群体网状、非纯线性，用户可在任意环节分享或放弃。
- **接触内容过度营销**：在认知教育阶段夹带过多推销，失去"中立感"，引发质疑与负向口碑，对高门槛产品尤甚。
- **价格/促销套路化**："先涨价再搞活动"等手段让用户感到被欺骗，延迟或终止转化；过度现金补贴养成"不送不用"的反向习惯，投入产出比低。
