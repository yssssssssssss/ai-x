# 心理物理学（阈限与信号检测论）

> 来源：神灯圈子·赵真真《心理物理学方法在用户研究中的应用（一）感觉阈限测量篇》 原文 http://xingyun.jd.com/shendeng/article/detail/52441
> 来源：神灯圈子·赵真真《心理物理学方法在用户研究中的应用（二）信号检测论篇》 原文 http://xingyun.jd.com/shendeng/article/detail/54961

> 一句话：把"物理刺激"与"主观感觉"之间的函数关系当作一把尺子，去量用户"刚刚好能感觉到""刚刚好能感觉到变化""能不能从噪音里识别出信号"的感知边界。
> 落点：`models/`。判据——它是一套贯穿全程的透镜（怎么定义"够用/卡顿/优惠""降价多少能被察觉""模型打标签准不准"都从它推导），拆开就讲不通。
> 边界：本文描述「用户的感知本身长什么样、边界在哪」；具体怎么测、怎么算阈限或 d'/β/ROC 属 `toolbox/`。

## 核心概念

- 一句话讲清：心理物理学（Psychophysics）研究**物理刺激与心理感受之间的函数关系**，为用户研究提供把主观感知量化的工具——核心是两件事：① 用**感觉阈限测量**找到"有/无""变/不变"的感知临界点；② 用**信号检测论（SDT）**把"感知能力"与"判断标准"分开来量化。
- 它要回答用户体验里那些"刚刚好"的边界问题，例如：
  - 手机消息提示音量调到多大时，用户会觉得"够用但不吵"？
  - 内容加载延迟控制在什么范围才不被感知为"卡顿"？
  - 界面元素的大小、颜色对比度达到什么标准才能被轻松感知？
  - 手机重量多少，用户会觉得"有质感"而不是"沉甸甸累手"？
  - 竞品价格 300 元，运营想小幅降价提升转化，降多少能让用户"刚刚好"感觉到优惠、又尽量省成本？
- 这些问题的答案也是评估技术、成本投入性价比的关键。访谈、问卷、可用性测试在此往往力不从心，需要这把更灵敏的"尺子"为设计决策找到数字锚点。

### 物理刺激 vs 心理感觉

讨论阈限测量前必须区分**物理刺激**（客观的、可用仪器测量）与**心理感觉**（主观的、因人或环境而异）：

- 温度感觉：9 月跳进郊外小河，入水一刹那觉得很凉，过一会儿就不那么凉了——河水的物理温度是一回事，主观感觉是另一回事。
- 重量感觉：1kg 棉花与 1kg 铅物理重量一样，但感觉铅更重；课堂演示中，大学生通常把 720g 的枕头判断为与 30–225g 之间某种重量的铅同样重。
- 优惠感：一件商品标价 99 元 + 运费 6 元 = 总支出 105 元，另一件相同商品直接标价 105 元且包邮，金钱数量相同，但用户会觉得后者更贵。

核心问题由此而生：**我们该如何测量这些由物理刺激引起的主观感觉？** 这正是心理物理学要解决、也是用户研究可借鉴的关键。

## 理论来源

- **感觉阈限 / 心理物理学**：先驱者 G．T．Fechner，通过对感觉强度与刺激强度数量关系的长期研究发展出测量感觉的基本方法。他给心理物理学下的定义是：**一门研究心身之间或心物之间的函数关系的精密科学**。一百多年来方法不断发展，但中心问题始终是物理量与心理量之间的数量关系。
  - Fechner 发现感觉强度的增长与刺激的增长并非 1∶1：一只铃在响，增加第二只铃的影响要比 10 只已在响的铃再加一只强烈得多；4～5 根蜡烛正发光时再加一根只造成微乎其微的差别，而原来只有 2 根时，加一根造成的影响就相当大。
- **信号检测论（Signal Detection Theory, SDT）**：信息论的一个重要分支。1954 年，美国心理学家 W.P. Tanner 和 J.A. Swets 将其引入人类知觉研究，推动心理物理学方法进入新阶段。

## 关键构念 / 维度

本模型含两大块：**① 感觉阈限测量** 与 **② 信号检测论**。

### ① 感觉阈限测量

感觉阈限：个体能刚刚感受到某个刺激存在、或刚刚能感受到两个刺激有差异时，所需的**最小物理刺激量**。分为两种：

**（一）绝对阈限（Absolute Threshold）：解决"有"或"无"的问题**

- **定义**：刚刚能引起感觉的最小刺激强度。感觉从"无"到"有"对应一系列由小到大的刺激——刺激小则感觉概率小，刺激大则感觉概率大，因此绝对阈限不是单一强度，而是一系列强度不同的刺激。故**绝对阈限定义为：有 50％ 的次数能引起感觉、50％ 的次数不能引起感觉的那一种刺激强度**。
- **核心价值**：帮产品确定功能或设计的**"存在性边界"**，确保关键信息能被可靠感知。
- **用户研究示例**：
  - 听觉设计："App 推送提示音需要多大音量（分贝），用户才能在嘈杂地铁里刚刚好听到？"（"存在"的临界点）
  - 视觉设计："图标尺寸需要多大，用户才能不费劲一眼注意到它？"（"可见"的临界点）
  - 性能体验："加载动画需持续多少毫秒，用户才意识到系统在处理而非卡死？"（"可辨识状态"的临界点）

**（二）差别阈限（Difference Threshold）：解决"变"和"不变"的问题**

- **定义**：刚刚能引起差别感觉的两个刺激之间的最小差异量，也称**最小可觉差（Just Noticeable Difference, JND）**。回答"需要改变多少，人们才看/听/感觉得出来？"。
- **核心价值**：指导**"改变的程度"**，寻找"临界变化量"。想让用户明显感知正向变化（如性能提升），变化量须**大于 JND**；想做负向调整（如隐形提价）而不引反感，变化量应**小于 JND**。
- **用户研究示例**：
  - 性能优化：页面加载从 1000ms 优化到 800ms，用户能感知"变快"吗？是否要到 700ms 才有体感？（"更快"的临界变化量）
  - 价格策略：100 元商品降多少（5/10/15 元）用户才明显感知"优惠"并提升购买意愿？（"更便宜"的临界变化量）
  - UI 调整：为过无障碍审核，按钮与背景对比度调到多少比率，用户才能轻松区分？（"更清晰"的临界变化量）
  - 内容策略：信息流推荐比例增加多少，用户才察觉并可能产生"内容太杂"的负面情绪？（"更多推荐"的临界变化量）

### ② 信号检测论（SDT）

SDT 的突破：能把个体的**"感知能力"**与**"决策标准"**区分开，分别用独立指标量化——解释用户如何在多种干扰中识别重要刺激，这不仅是感觉能力问题，更涉及用户的动机与期望。其原理是统计原理与最优决策原则，核心为：**两个分布、四种结果、两个指标、一条曲线**。

> 生活情境（贯穿下文）：你在嘈杂聚会上（音乐、谈话混杂），突然好像听到有人叫你的名字。你真的听到了吗（**信号** = 你的名字），还是背景噪音的随机组合让你"以为"听到了（**噪音** = 其他谈话/音乐）？

**（一）两个分布**

感官神经系统接收刺激时产生一个内部的"神经兴奋值/感觉强度"，是连续变量。只有噪音时（嘈杂声）形成一个正态分布（**噪音分布**），信号出现时（名字被叫出）形成另一个正态分布（**信号分布**）。两分布几乎总有重叠——有时噪音很"像"信号（有人说了个类似你名字的词），有时信号很微弱（叫得声音小）。

![](https://s3.cn-north-1.jdcloud-oss.com/shendengbucket1/2025-12-01-19-54W77pRHBQt54BjnTu.png)

**（二）四种结果**

按"信号是否真实存在"与"你如何反应"组合出四种结果：

![](https://s3.cn-north-1.jdcloud-oss.com/shendengbucket1/2025-12-01-19-54d16KL12O1xxkKrQ0t.png)

- **击中**：信号出现时用户报告"有"（有人叫你、你也听到）；击中率 = 击中次数 / 信号出现总次数
- **漏报**：信号出现时用户报告"无"（有人叫你、你没听到）；漏报率 = 漏报次数 / 信号出现总次数
- **虚报**：无信号只有噪音时用户报告"有"（没人叫你、你却以为有人叫）；虚报率 = 虚报次数 / 噪音出现总次数
- **正确拒绝**：无信号只有噪音时用户报告"无"（没人叫你、你也没听到）；正确拒绝率 = 正确拒绝次数 / 噪音出现总次数

![](https://s3.cn-north-1.jdcloud-oss.com/shendengbucket1/2025-12-01-19-5454GSFSjieRWsNBnf.png)

**（三）两个核心指标**

1. **辨别力 d'**：感官系统区分信号与噪音的**内在能力 / 客观敏感度**。噪声不变时，信号越强越易辨别，信号分布均值 MSN 与噪声分布均值 MN 的差越大、两分布重叠越少；辨别力强者 MSN 与 MN 的差大，弱者差小。故两分布均值的距离可作辨别力指标，即 **d'**。
   - **d' = Z(击中率) − Z(虚报率)**。d' 大 → 两分布重叠少、很容易区分（如安静图书馆里听名字）；d' 小 → 重叠多、很难区分（如摇滚演唱会里听细微耳语）；**d' = 0 意味着完全无法区分信号与噪音**。
   - d' 是相对纯粹的指标，不受判断标准变化影响——**无论标准是严是松，辨别能力本身不变**。

2. **反应倾向性——决策标准 β**：个体做出"有信号"反应所需的**证据强度 / 主观门槛**，易受动机、态度、信号出现概率等影响。以"是否回头"理解：
   - **低标准 / 宽松标准**（宁可错杀，不可放过）："听着有点像就回头看看"，"回头线"设得很低（约 30% 相似度就回头）。几乎不会错过任何一次真叫（**击中率很高**），但会因别人说"李明""绮晴"等不相干的话频繁回头（**虚报率很高**）。
   - **高标准 / 严格标准**（必须百分百确定）："听得非常清楚才回头"，"回头线"设得很高（须 90% 以上相似度才回头）。几乎不会出现没人叫却回头的尴尬（**虚报率很低**），但对方声音小或模糊时可能真没听见而错过（**漏报率很高**）。
   - **β = O(击中率) / O(虚报率)**，即信号分布上判断标准处的纵高与噪音分布上判断标准处的纵高之比。
     - β > 1：标准严格（需很强证据才说"有"）
     - β = 1：标准中立
     - β < 1：标准宽松（较弱证据就说"有"）

**（四）一条曲线**

**接收者操作特性曲线（ROC 曲线）**，又称等感受性曲线——曲线上各点对应相同的感受性，都是对同一信号刺激、在不同判断标准下的反应。判断标准不同会得到几组不同的击中率与虚报率；ROC 曲线即由这些击中率、虚报率转化的 Z 分数绘成，**横轴为虚报率、纵轴为击中率**。

![](https://s3.cn-north-1.jdcloud-oss.com/shendengbucket1/2025-12-01-19-54M12xfPV54ix7zeWu12.png)

## 如何作为透镜使用

- **问题定义阶段**：当遇到"多少才够？""多小才能被发现？""多大变化才被注意？""用户能不能从噪音里把信号识别出来？"这类问题时，先判断它属于哪一类——
  - "有/无"边界 → 绝对阈限；"变/不变"边界 → 差别阈限 / JND；
  - 当不同用户的**判断标准**可能不同（有人宽松、有人严格），需要把"感知能力"与"判断标准"剥离时 → 用 SDT。
- **采集阶段**：把刺激设计成可控的物理量序列（音量/延迟/对比度/价格等）让用户做"有无/异同"判断；阈限测量可用最小变化法（极限法）、恒定刺激法、平均误差法（详见下节）；SDT 类研究需同时记录信号试次与噪音试次下的反应，以便算出击中率/虚报率。
- **分析 / 结论阶段**：
  - 阈限类：绝对阈限取"50% 能引起感觉"的刺激强度；差别阈限取最小可觉差（JND），用其指导"提升要大于 JND、隐形调整要小于 JND"。
  - SDT 类：算 d' 看辨别力（与标准无关）、算 β 看判断标准松紧、画 ROC / 算 AUC 综合评估，并结合"漏报/虚报"的业务成本选阈值。

### 应用案例：SDT 在大模型标签校验中的应用

以"自律健身族"标签为例：

**（一）评估大模型的判别能力**（最核心应用）——回答"大模型区分'自律健身族'与'非健身族'的能力到底多强"。
- 指标：**d'**，衡量模型判别能力的黄金指标。d' 越高，模型越能清晰把两类用户分开；d' 接近 0，说明模型基本在瞎猜。
- 方法：① 随机抽一批用户样本，让大模型预测每个用户是否为自律健身族；② 用研对同一批样本**盲审**（不知模型结果下独立评估）；③ 将模型预测与用研判断对比，计算 d'。

**（二）分析与优化模型的判断阈值**——大模型为每个标签输出 0–1 的置信度分数，需决定阈值。
- 指标：① ROC 曲线（绘不同阈值下击中率/虚报率，AUC 综合评估性能）；② 成本效益分析（明确"漏报""虚报"的业务成本）。
- 方法：
  - 场景 A（追求标签纯净度 / 高精度）：确保打上标签的用户 100% 真材实料；提高判断标准 β、置信度阈值设很高（如 0.9）；结果虚报很少但漏报增多；适用于高价值、高精准营销或种子用户小范围测试。
  - 场景 B（追求标签覆盖率 / 高召回）：尽可能网罗所有潜在用户；降低判断标准、阈值设较低（如 0.3）；结果漏报很少但虚报激增；适用于信息流内容推荐"广撒网"，误推成本也较低。

**（三）指导人工校验的资源分配**——人工校验昂贵，SDT 告诉你把人力集中在哪儿；通过分析被"虚报""漏报"的案例总结模型易混淆的模式，反过来为模型提供高质量反馈和训练数据。
- 指标：调整判断标准 β，在"虚报"与"漏报"间取得平衡。
- 方法：
  - 高置信区间（>0.9）：模型用极严格标准、几乎可肯定"信号"存在，在此投人力相当于复核"正确拒绝/击中"，投入产出比很低。
  - 低置信区间（<0.1）：模型用极宽松标准、几乎可肯定"噪声"存在，在此投人力相当于复核"正确拒绝/漏报"，同样低效。
  - 不确定区间（0.4–0.6）：判断标准最模糊、信号与噪声混淆程度最高（d' 在此子集最低）；按 SDT 思想，最大价值在于提升这个区域的判别力，方式就是投入人工判别这个"金标准"检测器。

## 与方法 / 分析的衔接

- **阈限测量的方法与程序**（"怎么测"，具体可查《实验心理学》）：
  1. **最小变化法（极限法）**：将刺激按"递增""递减"系列交替呈现，要求用户报告是否感受到刺激（测绝对阈限）或是否感受到差异（测差别阈限）。以测音高绝对阈限为例：
     - ① 测前先将刺激量分为递增、递减两种系列。
     - ② 递增系列起点安排在被试基本觉察不到的强度内，递减系列起点安排在基本能觉察到的强度内。
     - ③ 每个系列从起点沿递增/递减方向依次呈现，增减幅度尽可能小以探测转折点；每次呈现后要求被试报告：有感觉记"＋"、无感觉记"-"、不能肯定记"？"（"？"的意义与其前一个反应相反，如在第（二）系列中"？"表示有感觉，在第（五）与第（九）系列中"？"表示无感觉）。
     - ④ 递增系列只要被试第一次报告"有"即停止，递减系列只要第一次报告"无"即停止；两系列交替呈现，起点可随机选择，直到反应发生变化为止。
     - ⑤ 计算分两步：先算被试在每个系列上的阈限（反应转折点对应的两个刺激量的中点），再求所有系列阈限的均值，该均值即绝对阈限。

     最小变化法测音高的绝对阈限：

     ![](https://s3.cn-north-1.jdcloud-oss.com/shendengbucket1/2025-10-09-19-28kqfIseuMqpuxY280.jpg)

  2. **恒定刺激法**：① 刺激通常 5～7 个，实验过程中维持不变；② 最大强度大到被感觉概率约 95%、最小强度小到约 5%；③ 各刺激间距相等、**随机呈现**（与最小变化法不同），每个刺激呈现次数相等；④ 最后用**直线内插法**计算感觉阈限。
     - 示例（手掌两点阈测绝对阈限）：选最大/最小刺激 12mm、8mm，按 1mm 间距确定其余为 9/10/11mm，每个随机呈现 200 次，每次呈现后被试回答"两点"还是"一点"。结果相距 10mm 有 29% 被判为两点、11mm 有 66% 被判为两点，则 50% 判为两点的刺激必在 10mm 与 11mm 之间。
  3. **平均误差法**：① 实验者规定某一刺激为标准刺激（如界面固定亮度）；② 要求被试自己调节另一刺激使其在感觉上与标准相等（直到两者亮度看起来一样）；③ 重复多次记录每次调整的最终值；④ 计算这些调整值的平均误差作为差别阈限的估计值（平均误差与差别阈限成正比，故可用其表示差别感受性）。求平均误差有两种：一是每次调节结果与标准刺激之差的绝对值平均，二是每次调节结果与主观相等点之差的绝对值平均。
- 概念落到可操作分析、出图（如画 ROC、用 d'/β 比较多个判别器或多组用户）时看：`methods/toolbox/analysis/...`；与量化感知/偏好比较相关的衔接可参考对偶比较法与量表编制（见下方关联）。

## 局限与误用

- **传统阈限测量混淆了"能力"与"标准"**：即便不同用户对某一刺激的实际感知能力相同，作判断时采用的标准也可能不同（有人宽松、有人严格）。仍用传统心理物理学方法时，测量结果可能因主观判断标准不同而产生偏差。例如用恒定刺激法测差别阈限，结果往往受被试自信心或谨慎程度的干扰。在需要充分考虑用户判断标准差异的项目中，应改用 SDT，把"感知能力（d'）"与"决策标准（β）"分别量化。
- **不要用一次"是否察觉"就断定谁更敏锐**：甲说"有人敲门"、乙说"没听见"，不能据此认定甲听觉更敏锐——两人可能敏感度相同，只是甲"冒进"（隐约有声就报告"有"以避免遗漏）、乙"保守"（除非十足把握否则不报告以减少误判）。同理夜里"手机好像响了"，差异可能来自期望（甲在等消息"宁可信其有"、乙默认"这个点多半是广告"），而非感官强弱。
- d' 与 β 必须区分：d' 不受判断标准变化影响，β 才反映标准松紧；混为一谈会把"标准差异"误读成"能力差异"。
