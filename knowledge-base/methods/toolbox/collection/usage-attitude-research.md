# U&A 使用习惯与态度研究

> 一句话：通过结构化问卷系统测量目标人群对一类产品 / 品牌的**使用习惯（Usage）**和**态度（Attitude）**，输出市场状况、目标消费者、竞争对手三方面的可行动信息。
> 落点：`methods/toolbox/collection/`。本方法**唯一出处**——scenarios / skills 引用本文件，不复制。
> 边界：U&A 是定量方法，主要载体是问卷；与定性的访谈类方法（[interviews](interviews.md)、[switch-interview](switch-interview.md)、[job-interview](job-interview.md)）配合使用。

## 何时用 / 不适合

### 适用

- **战略规划场景**：营销战略、产品战略、品牌战略——尤其是 **TO C 类企业**做面向广大消费者的战略规划。
- **典型行业**：快速消费品、家电、食品/饮料、化妆品/洗涤品、日用品（U&A 在快消最成熟）；耐用消费品也常用。
- **典型营销决策问题**（U&A 直接服务于这些）：
  1. 为现有产品或新产品**寻找市场机会**；
  2. 有效地**细分市场**，选择目标市场和确定产品定位；
  3. 制定**营销组合策略**；
  4. **评价企业的市场营销活动**。

### 不适用

- **TO B 类企业的公司级战略**——用户研究往往淹没在宏观环境分析中，U&A 价值有限。
- 仅做需求洞察/机会发现的小样本探索（用定性访谈方法）。
- ⚠️ 待补充(缺源)：原文未明确"不适用"清单。

## 前置准备

- **招募与样本**：定量样本（一般几百到几千），按 [[sampling]]（`methods/standards/sampling.md`）做。
- **物料**：标准问卷（核心结构见下方"执行步骤"）、卡片（让受访者按卡片选答案）。
- ⚠️ 待补充(缺源)：原文未给出最小样本量阈值。

## 执行步骤

### Step 1 · 明确战略问题与营销决策

把 U&A 要回答的营销决策问题先写清楚（市场机会 / 细分定位 / 营销组合 / 评价活动 这四类之一或组合）。

### Step 2 · 设计三方面调查内容

U&A 没有统一固定的模型框架，但**总少不了三方面分析**：

#### 2.1 消费者特征（包括五类）

| 特征类 | 内容 |
|---|---|
| **人口统计特征** | 年龄、性别、地域、职业、学历等 |
| **社会统计特征** | ⚠️ 待补充(缺源)：原文标题"二、社会统计特征"具体内容缺失 |
| **经济统计特征** | 收入是核心——不同收入阶层在购买决策、频率、数量上存在差异。高收入注重品牌/品质/服务，低收入注重价格/实用 |
| **媒介消费/接触情况** | 电视、网络、报纸、杂志、社交媒体接触习惯 |
| **生活习惯与价值取向** | ⚠️ 待补充(缺源)：原文标题但具体内容缺失 |

> 注意：**人口统计标签是静态标签，粒度偏粗**——需与其他维度结合才有真正业务价值。

#### 2.2 消费者态度（针对特定产品/服务的长期性认知评价、情感体验、行动倾向）

四个核心内容：

1. **产品/品牌知觉与偏好**——基于外观、功能、宣传形成的初步认知。
2. **产品/品牌态度与购买意愿**——积极态度→购买意愿；消极态度→抑制购买。
3. **企业/产品评价**——质量、价格、服务 + 品牌形象、社会责任感。
4. **生活态度与价值取向**——预示对新产品、新概念、新功能的接受程度（追求时尚潮流→愿试新；注重环保→选绿色产品）。

#### 2.3 消费者行为（五个层面）

##### A. 购买决策过程（5 步）

⚠️ 待补充(缺源)：原文明确为"确认问题、信息搜集、方案比较、购买决策、购后行为"——已可读，已捕获。

##### B. 产品/品牌购买参数

时间 / 地点 / 方式 / 价格 / 数量 / 产品或品牌的具体选择。

##### C. 产品/品牌使用习惯与消费行为

使用时间、场所、数量、频次、使用者、品牌转换情况。

##### D. 产品/品牌使用/转换/停用的原因

| 行为 | 典型原因 |
|---|---|
| **使用** | 满足需求 / 价格合理 / 质量可靠 |
| **转换** | 对现有不满 / 发现更好替代 / 受推荐 |
| **停用** | 不再满足 / 出现质量问题 / 价格过高 / 个人原因 |

##### E. 产品/品牌的信息沟通渠道与方式

- **渠道**：广告 / 促销 / 社交媒体 / 口碑推荐
- **方式**：文字 / 图片 / 视频 / 音频

### Step 3 · U&A 问卷标准核心问题模块

> 不同日用消费品的标准问卷应作适当增减。**核心问题模块**：

| 模块 | 内容 |
|---|---|
| **品牌和广告认知** | 提示前认知 / 提示后认知 / 广告记忆 |
| **产品使用情况** | 是否使用 / 主要品牌 / 使用频率 / 使用方法 |
| **购买情况** | 主要购买渠道 / 购买频率 / 单次购买金额 / 决策者 |
| **对品牌的态度** | D1 品牌使用情况句子选择（卡片） / D3 品牌评价因素 / D4 各品牌在每个因素上的 10 分评分 / D6 理想品牌形象 |
| **对产品的态度** | E1 关于产品态度的语句 5 分制同意度量表 |

## 关键技巧与注意

### 三方面分析必须配齐

特征 → 态度 → 行为构成一条因果链。**只研究行为不研究态度，找不到为什么；只研究态度不研究行为，没法验证。**

### 重度使用者要单独刻画

重度使用者的人口统计特征 / 行为 / 态度往往与一般用户差异巨大，是营销资源应当倾斜的核心目标。

### 与定性互补

U&A 给"是什么、有多少"，定性访谈给"为什么"。建议先 U&A 摸出全貌，再用定性深挖关键差异。

## 常见坑

- **问卷过长** —— 五大模块全填会让问卷臃肿，应**根据本次决策问题做模块取舍**。
- **态度问题做成态度量表 ≠ 真实行为** —— 态度题答的是"嘴上说"的，要和"行为题"交叉验证。
- **品牌列表不全** —— 品牌引力 / 形象 / 优势弱点的对比，必须**包含所有主要竞争品牌**——遗漏关键竞品会让结论失真。
