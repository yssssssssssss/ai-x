# 焦点小组（Focus Group）

> 一句话：由受过专业训练的主持人，集中在一个或一类主题，用结构化方式**同时访谈多名用户（一般 6~8 人）**，揭示目标用户的经验、感受、态度与愿望，并努力客观呈现其背后理由的研究方法。
> 落点：`methods/toolbox/collection/`。本方法**唯一出处**——scenarios / skills 引用本文件，不复制。

## 核心要素

| 要素 | 说明 |
|---|---|
| **焦点** | 集中在一个或一类主题，结构化方式揭示用户经验、感受、态度、愿望 |
| **小组** | 多人同时访谈，一般为 **6~8 人**（最适宜） |
| **主持人** | 引导整场访谈顺序进行，启发用户思考回答；行业经验丰富，对产品有足够了解 |
