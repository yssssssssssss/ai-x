# 品牌命名研究与测评

> 来源：神灯圈子·赵真真《品牌符号管理之一：品牌命名的原则、策略与流程》 原文 http://xingyun.jd.com/shendeng/article/detail/46569

> 一句话：围绕品牌命名展开的研究与测评方法——既给出命名应遵循的原则与策略，更聚焦候选名称如何被验证（问卷 / 电话 / 输入法 / A/B 实验），帮助从一批候选中选出"好名字"。
> 落点：`methods/toolbox/collection/`。命名测评方法本体在此**唯一出处**，scenarios/skills 引用而非复制。

## 何时用 / 不适合

- 适用：
  - 新品牌 / 新品类 / 新产品需要从 0 命名，需要从一批候选名称中筛选并验证最优选择。
  - 已有品牌名待评估其吸引力、易记性、联想是否符合预期。
  - 需要为命名决策提供量化或定性证据（吸引力、理解力、记忆率、购买意图等指标）。
- 不适用：
  - 仅做品牌口号、品牌标识等其它品牌符号管理（原文计划另篇分享，本篇仅覆盖命名）。
  - 名称已由法律 / 战略硬约束唯一确定、无候选可比时，测评意义有限。

## 前置准备

- 招募与样本（遵循 `methods/standards/sampling.md`）：
  - 测评对象应为该品牌的**潜在消费者**；电话测试法、问卷法均面向潜在客户回收感知。
- 物料（候选名称 / 问卷 / 脚本 / 触点）：
  - 经收集筛选后进入"决赛圈"的若干候选名称。
  - 问卷量表：按测量目的选取下列一项或多项指标的题目。
  - 电话测试脚本：告知名称→请对方写出→隔 X 日回访探测。
  - A/B 实验所需的线上触点（曝光 / 点击 / 转化埋点），若产品或服务有线上入口。

## 执行步骤

品牌命名整体流程分四步，研究/测评集中在第 3 步，前两步为其提供候选输入：

1. **前期调查**：考虑外部因素与内部环境。对外分析竞争对手、行业特征、文化氛围等；对内思考品牌定位、产品形态、服务特色等，进而做出策略性讨论。
2. **收集筛选**：尽可能多地收集候选名称，罗列越多越好。两种收集方式——
   - 内部头脑风暴，拓展想象空间，互相激发出多种命名；
   - 从消费者侧收集（"高手在民间"），通过问卷方式收集消费者对品牌的命名。
   随后内部初步筛选，挑出最佳的几个候选命名进入"决赛圈"。
3. **测评验证**（关键阶段）：对候选名称用更严谨的方式验证并筛选出最佳选择。常用四种验证方法：
   - **① 问卷调研法**：通过问卷收集潜在消费者对品牌名称的感知，按测量目的选择以下一项或几项关键指标进行测试——
     - 吸引力：品牌名称是否引起了潜在客户的注意？
     - 理解力：品牌名称在表达什么意思，是否容易理解？
     - 可靠性：品牌名称所传达的信息是否可信？
     - 发音容易：品牌名称发音容易吗？
     - 记忆容易：品牌名称容易记忆吗？
     - 购买意图：品牌名称是否会激发人们购买产品？
     - 相关性：品牌名称是否反映了受访者的需求和愿望？
     - 独创性：这个品牌名称与其他品牌名称有何不同？
   - **② 打电话测试法**：打电话给潜在消费者，把名字告诉他们。先看他们能否直接写出来；再隔几天试探性回访，看是否还记得品牌名。收集与统计消费者隔 X 日提及率，本质上评估品牌名称的**易记程度**，收集消费者的**回忆率与再认率**。
   - **③ 输入法测试法**：用搜狗输入法去输入备选品牌名，看是否能"一次连打"出来。若是三个字的品牌名需分别打出三个字，基本都不是好名字。其原理是模仿消费者"找你"的过程——若打出品牌名都很费劲，意味着消费者找到你的难度和成本较高。
   - **④ A/B 实验测试法**：若产品或服务的品牌名称有线上触点，可通过 A/B 测试评估**曝光 / 点击 / 转化**等用户行为指标作为参考。
4. **注册认证**：确保品牌命名的可行性。确定最终品牌名后，出于经营战略考量，会同时注册多个类别的商标，进行自我保护。

## 关键技巧与注意

- **测评是从一批候选里选优，而非验证单一名称**：收集阶段越多越好，测评阶段用客观方法在决赛圈候选间做差异化比较。
- **指标按目的取舍**：问卷的八项指标不必全测，依测量目的选一项或几项；如目标是传播效率重点测易记/发音，目标是转化则关注购买意图。
- **多法互证**：易记程度可由电话测试（回忆率/再认率）与输入法测试交叉印证；线上有触点时再以 A/B 行为数据校验主观自评，降低单一方法偏差。
- **命名需综合多学科**：综合营销学、语言学、心理学、文化、法学、美学等，测评指标的设计应回扣命名原则（见下「命名原则与策略（设计测评指标的依据）」）。

## 命名原则与策略（设计测评指标的依据）

> 命名原则/策略是测评指标的来源——验证一个名称是否"好"，本质是检验它在多大程度上满足这些原则。品牌营销理念部分从简，详见原文。

**命名的重要性（简述）**：品牌命名是品牌建设的第一步、是最重要的品牌要素（"品牌的心脏"）；研究人员认为新品牌能否获消费者青睐有约 40% 因素来自品牌名称；品牌名称在很大程度上影响品牌联想，并比其它营销计划更持久。

**命名的六项原则**（测评指标可逐条对应）：

| 原则 | 含义要点 | 对应可测维度 |
|---|---|---|
| 1. 受法律保护 | 能注册才真正属于自己，防"山寨/搭便车"（如阿里注册"阿里妈妈/爸爸/姐姐/奶奶"） | 注册可行性（流程第 4 步） |
| 2. 易发音、记忆 | 减少生僻字/多音字；注意字数与发音结构。叠音增强听感：AA（滴滴/脉脉/探探）、ABB（拼多多/货拉拉/娃哈哈/香飘飘）、ABAB（哔哩哔哩/呷哺呷哺）；字数越少越易记，2–4 字居多（耐克/宝马/京东/淘宝/星巴克/可口可乐），5 字以上较少（施华洛世奇） | 发音容易、记忆容易 |
| 3. 新颖独特 | 在饱和市场中脱颖而出，提高可识别性与记忆度（特斯拉、ofo 小黄车） | 独创性 |
| 4. 暗示功能属性 | 名称与产品功能/特点相吻合（飘柔=飘逸柔顺、汰渍=淘汰污渍、小米=亲民性价比） | 理解力、相关性 |
| 5. 启发联想 | 传达美好寓意，产生积极联想（海信＝海纳百川/信诚无限，Hisense＝High+Sense） | 吸引力、可靠性 |
| 6. 自由延伸 | 具传承性与适应性，可拓展到其它领域（"滴滴打车"→"滴滴出行"以涵盖代驾等业务） | （战略层，影响命名取舍） |

**命名的策略**（候选名称的产生方向，供收集阶段发散）：

- **（一）以产品带给消费者的利益命名**
  - 功效性命名：
    - 定位命名（暗示特征/场合/品类）：品类特征（农夫山泉）、原料成分（全棉时代）、利益功效（飘柔/舒肤佳/金嗓子）。
    - 数字命名（借数字联想、增差异化）：999、7‑11、1688。
    - 时空命名（历史渊源增文化内涵与正宗感）：道光廿五、年三十、张一元（"一元复始，万象更新"）。
  - 情感性命名：
    - 动植物命名（移情其品质特征，简单好记）：七匹狼（勇猛睿智）、阿里"动物园"（天猫/飞猪/菜鸟/蚂蚁/神马/千牛/虾米音乐）。
    - 价值命名（凝练价值追求）：远大、健民、同仁。
- **（二）以品牌来源命名**
  - 人物命名（借身份威望与崇拜心理）：创始人/发明者（李宁/王致和/胖东来）、历史人物（杜康/伊丽莎白）、常用称呼（爸爸糖）、身份职称（吾皇/李医生/牙护士）。
  - 产地命名（借原产地效应）：蒙牛、鄂尔多斯、青岛啤酒、贵州茅台、沪上阿姨。
- **（三）以目标市场/顾客命名**：以目标客群作名，使其产生认同感并暗示年龄/性别等（好太太口服液、好孩子）。
