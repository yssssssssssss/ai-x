# methods/toolbox/analysis/ · 分析技术

> 能**跨主题复用**的分析手段的本体：IPA 满意度×重要性矩阵、conjoint、因子分析、亲和图、旅程图…
> 何时进：想知道「某个分析手段怎么算 / 怎么出图 / 怎么解读」。
> 入选判据：这套手段能脱离原研究被单独拿去用（这正是它该从某个方法里被「拆出来」的理由）。
