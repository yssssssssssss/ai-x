# 贝叶斯平台间品类优势对比

> 来源：神灯圈子·贺兆铎《基于贝叶斯法则实现平台间品类优势的有效对比》 原文 http://xingyun.jd.com/shendeng/article/detail/46686
> 本文原作者李妍，写于 2023.01.06

> 一句话：在大促等多平台品类购买调研中，用条件概率（贝叶斯法则）修正平台规模差异，得出真正可比的平台间品类优势。

## 何时用
- 适合的问题类型：大促/日常用研中问到用户在不同平台（本品 vs 竞品）的品类购买情况，想对比各平台分别在哪些品类上有市场优势、差距在哪里、有多大。
- 与近似技术的区别：直接拿"某平台用户中购买该品类的比例"做横向对比是常见误区——它忽略了平台市场规模的影响。本方法的关键是区分**单平台内部品类优势**与**平台间品类优势**两类问题，后者必须做条件概率修正。

### 误区：忽略前提条件导致的错误判断
假如根据调研问卷，京东用户有 30% 的人在大促购买生鲜，而淘宝用户只有 26% 购买了生鲜，能否得出"大促期间更多人来京东买生鲜、京东生鲜有明显品类优势"？

其实不然。"在京东购买了生鲜"和"买生鲜的人是在京东买的"是两个完全不同的事情。假如京东只是一个小型生鲜网购平台，在该平台购买生鲜的用户比例肯定很高；但放之全市场，由于用淘宝的人更多，虽然只有 26% 的淘宝用户购买了生鲜，市场上生鲜品类售卖表现好的平台依然可能是淘宝。做平台优势对比时，一定要注意根据前提条件进行比例修正。

## 输入数据要求
- 多平台品类购买的问卷数据：通常包含两类题——用户在哪些平台购买（得到各平台购买用户比例），以及在各平台分别购买了哪些品类。
- 需可在**单样本**层面合并不同平台的购买品类情况，以便对品类做去重 count，得到单样本的全市场品类购买情况（用于计算 P(Bj)）。

## 分析步骤

### 1）贝叶斯法则
贝叶斯法则由英国数学家贝叶斯（Thomas Bayes）于 1763 年提出，用来描述两个条件概率之间的关系（即 P(A|B) 和 P(B|A)）。通常情况下，事件 A 在事件 B 发生的情况下出现，和事件 B 在事件 A 发生的情况下出现的概率是不同的，这是因为事件 A 和 B 发生的概率本身就不一样。

![](https://img12.360buyimg.com/ling/jfs/t1/31001/12/18947/39509/638deeffEfe7c898a/c54fac21480dfb9b.png)

几个概念：
1) $P(A)$：在不考虑事件 B 的情况下，对事件 A 发生的基本概率判断，称为 A 的"先验概率"；
2) $P(A|B)$：在事件 B 发生后，基于事件 B 对事件 A 发生概率的重新评估，称为 A 的"后验概率"；
3) $P(B)$：在不考虑事件 A 的情况下，对事件 B 发生的基本概率判断，称为 B 的"先验概率"；
4) $P(B|A)$：在事件 A 发生后，基于事件 A 对事件 B 发生概率的重新评估，称为 B 的"后验概率"。

由乘法法则易知 $P(A∩B) = P(A) \times P(B|A) = P(B) \times P(A|B)$，贝叶斯公式即为该式的变形：

$$P(A|B) = P(B|A) \times P(A) / P(B)$$

### 2）事件命名与问卷数据
在分析前先对事件命名：事件 $A_i$ 为"在 xx 平台购买"，事件 $B_j$ 为"网购了 xx 品类"。

![](https://img14.360buyimg.com/ling/jfs/t1/163756/38/29799/16407/63b3c42cFcc58a5f2/6704876e0d299610.png)

假设问卷回收结果如下图所示，则可知在甲平台购买商品的用户比例 $P(A_1) = 77\%$，在甲平台购买了食品酒水的比例 $P(B_1|A_1) = 47\%$，以此类推。

![](https://img12.360buyimg.com/ling/jfs/t1/172600/11/30157/60035/63b3a872F3672dfc6/7efd3665d73f8772.png)

### 3）先分清谁是前提条件 / 先验事件
对比平台品类优势，首先要搞清楚谁是前提条件 / 先验事件：

**a. 看单平台内部的品类优势**——先验事件是"用户来 XX 平台购买了商品"，其次才是购买了什么，公式表示为 $P(B_j|A_i)$，即来 xx 平台购买的人中有多少人买了 xx 品类。"在甲平台购买了生鲜"的比例就是 $P(B_5|A_1) = 27\%$（查上表可知），以此类推。

由此：甲平台购买量大的品类主要是食品酒水、鞋靴服饰与美妆个护；乙平台表现好的品类主要是食品酒水、美妆个护、生鲜与鞋靴服饰；丙平台表现较好的则为食品酒水与家清纸品。

![](https://img14.360buyimg.com/ling/jfs/t1/146228/17/31812/54178/63b3aa25F6ac2b667/4ef1f6b73dd5dfa2.png)

**b. 对比平台间的品类优势**——先验事件是"用户购买了 xx 品类"，其次才是分辨在哪个平台购买，这样才能把不同平台拉通在同一起点上、避免平台规模的影响。公式表示为 $P(A_i|B_j)$，即购买了 xx 品类的人中有多少人是在 xx 平台买的。"买生鲜的人是在甲平台买的"比例就是：

$$P(A_1|B_5) = P(B_5|A_1) \times P(A_1) / P(B_5) = 27\% \times 77\% / 54\% = 38\%$$

以此类推可得其他品类结果（如折线图所示）。

> 注 2：$P(B_5)$ 是什么、如何算？$P(B_j)$ 指网购了 xx 品类的用户比例，可通过合并单个样本在不同平台的购买品类情况、并对品类做 count 去重，得到单样本的全市场品类购买情况。$P(B_j) = $ 购买了 $B_j$ 品类的样本 / 总样本量。

## 结果解读
- **单平台内部（$P(B_j|A_i)$）**：回答"该平台来购物的人，最爱买哪些品类"，用于看一个平台自身的品类结构。
- **平台间对比（$P(A_i|B_j)$）**：回答"买某品类的人，更多流向了哪个平台"，扣除规模后才是真正的平台品类竞争力。
- 案例结论：经修正后，大促期间平台间对比来看，甲平台在鞋靴服饰、美妆个护方面表现出较强优势，乙平台则在手机数码、家电方面优势突出，丙平台优势主要在家清纸品类。

![](https://img12.360buyimg.com/ling/jfs/t1/186141/36/31892/103785/63b3a915F589d4bef/d1f824966135a7f2.png)

## 可视化 / 出图
- 单平台内部品类优势：各平台分品类的 $P(B_j|A_i)$ 条形 / 表格。
- 平台间品类优势：以品类为横轴、各平台修正后 $P(A_i|B_j)$ 为多条折线，便于看每个品类由哪个平台主导。

## 工具 / 实现
- 仅需问卷比例数据与四则运算即可套用贝叶斯公式；$P(B_j)$ 需在单样本层面对全市场品类购买做去重 count（Excel / SPSS / Python 均可）。

## 局限与误用
- **最常见误用**：直接用 $P(B_j|A_i)$（某平台用户中买该品类的比例）做平台间横向对比，忽略平台市场规模，导致"小平台品类比例虚高"的错误结论。平台间对比必须改用 $P(A_i|B_j)$。
- 选错先验事件就会答错问题：先想清楚要回答的是"单平台内部"还是"平台间"，再决定用 $P(B_j|A_i)$ 还是 $P(A_i|B_j)$。
