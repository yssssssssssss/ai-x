# TURF 分析

> 来源：神灯圈子·顾小妍《TURF分析：如何用最少的资源覆盖最大的市场》 原文 http://xingyun.jd.com/shendeng/article/detail/46511

> 一句话：用最少的资源（渠道/产品）覆盖尽可能多的不重复用户，通过计算备选组合的累积不重复到达率，找出覆盖最多用户的组合。
> 落点：`methods/toolbox/analysis/`。判据——这套手段能脱离原研究被单独复用（媒体渠道组合、产品线组合等任何"组合最大化覆盖"问题均可用）。

## 何时用
- 适合的问题类型：我们总是希望用最少的资源覆盖尽可能多的市场用户。典型场景包括：
  - 产品刚上线急需提升知名度，但不知如何选择媒体投放渠道，以最少的投入触达最多的用户。
  - 品牌进行整合营销传播后，想了解各个媒体渠道触达用户的情况，评估每个渠道的触达效率，为以后的营销传播计划提供依据。
  - 计划在原有产品线基础上新推出一个产品，想知道新产品是否有助于提升产品组合的整体市场份额。
- 与近似技术的区别：TURF 研究的是"累积"和"不重复"的数值——它关注组合后去重的覆盖率，而不是单个渠道/产品的触达率。因此触达率最高的几个渠道两两叠加未必是最优组合（用户重合度高会被去重抵消）。

## 输入数据要求
- 需要一份用户调研数据，记录每个用户被哪些备选项（渠道 / 产品）触达或使用，通常来自多选题。
- 每个备选项编码为多选题选项：选中 = 1，未选 = 0（用于 SPSS 的"正响应值"）。
- 备选项即待组合的对象集合（如各媒体渠道、各候选产品）。

## 分析步骤
TURF 分析通过计算所有备选组合的不重复累积值，找出覆盖最多用户的组合。

**累积不重复到达率的原理**

以某公司产品 A 和产品 B 为例：绿色圆圈表示产品 A 的用户占市场的 50%，黄色圆圈表示产品 B 的用户占市场的 30%。因为有一部分用户同时使用产品 A 和产品 B，直接加总会高估用户占比，因此需要去重。

![](https://img12.360buyimg.com/ling/jfs/t1/87910/33/157/13816/5da81bf1E79f1ae91/00426659641824c6.png)

该公司整体用户占比应为：**50% + 30% - 10% = 70%**。

将 50% 和 30% 加总，是为了求"累积"的数值；减去重合的 10%，是为了得到"不重复"的数值。这就是"累积不重复"的含义。

**计算流程**

TURF 分析按照组合中的备选项数量分步进行：

1. 按组合内项目个数分步：先算所有 2 项组合，再算所有 3 项、4 项组合……（若总项目数为 N，则 N 项组合的累积不重复到达率必然 = 100%，无需计算）。
2. 在每一档数量下，列出所有可能的两两 / 三三……组合，逐一计算每个组合的累积不重复到达率。
3. 在每一档内选取累积不重复到达率最高的组合作为该档最优解。
4. 对比各档结果，结合资源预算确定最终组合；预算减少时可"有的放矢"地砍掉对覆盖率提升功效最小的项目。

## 结果解读

**案例 1：如何选择外投渠道组合才能覆盖最多的用户**（本文数据均为虚构）

某公司新产品做了一波整合营销传播，在多个渠道外推，并调研用户从哪些渠道知道该新产品。原始触达数据如下：

![](https://img12.360buyimg.com/ling/jfs/t1/73916/40/13175/7415/5da8242bE706b26d4/5b86e4603e6ea3b1.png)

从单渠道数据看，微博触达最多用户，其次是微信朋友圈和公众号。但这并不能说明"微博 + 微信"组合覆盖率最高。

先看 2 个渠道的组合（两两组合分别计算累积不重复触达率后对比）：

![](https://img13.360buyimg.com/ling/jfs/t1/94417/37/172/26917/5da82888Ed48e5bb1/bb77720d585202dd.png)

只投 2 个渠道时，触达最多用户的最优选择不是微博 + 微信朋友圈（累积不重复触达 78%），而是微博 + 短视频 APP（累积不重复触达 85%）。因为微博和微信朋友圈触达的用户重合度高，而微博和短视频 APP 触达的用户重合度低。

继续按相同方法计算 3 个、4 个渠道组合，最终结果如下：

![](https://img12.360buyimg.com/ling/jfs/t1/77383/17/13154/12066/5da82e3bE08694f43/723bf44e1e43d3a8.png)

微博 + 短视频 APP + 地推的组合已能触达 95% 用户；该组合中短视频 APP 和地推的覆盖效果优于微信朋友圈和公众号。当预算减少时，可砍掉对触达率提升功效最小的渠道（本案例中是微信公众号和微信朋友圈）。

结论：要触达最多用户，不能只选触达率最高的几个渠道，而要通过 TURF 分析研究如何组合。

**案例 2：如何选择产品组合以扩大整体用户规模，减少产品间蚕食**

某公司现有产品 A，计划推出产品 B 或产品 C 以占领更多市场份额。用 TURF 分析判断哪个产品能占领更多份额、同时蚕食更少。

![](https://img30.360buyimg.com/ling/jfs/t1/81917/37/13198/16066/5da83328Ec7853cab/d4350e5dc05e9678.png)

![](https://img14.360buyimg.com/ling/jfs/t1/74342/24/13211/17158/5da83332Eaa921d4a/457d5f93bb9da3d5.png)

绿色圆圈表示产品 A 目前用户比例，黄色圆圈表示潜在用户比例。产品 B 潜在用户比例 20%，产品 C 潜在用户比例 30%，产品 C 潜在用户群体更大。但从获取更大市场份额（更多用户）的标准看，应通过 TURF 分析判断上线哪个产品：

- 若上线产品 B，可能获取的用户比例：50% + 20% - 5% = 65%
- 若上线产品 C，可能获取的用户比例：50% + 30% - 20% = 60%

也就是说，产品 B 对扩大用户规模的贡献是 15%（65% - 50%），产品 C 的贡献是 10%（60% - 50%）。潜在用户更多的产品 C，因蚕食更严重，反而对整体规模贡献更小。

## 可视化 / 出图
- 用文氏图（圆圈交叠）展示各项目用户占比及重合部分，直观呈现"累积"与去重逻辑（见上方原理图与案例 2 图）。
- 用组合对比表展示每档数量下各组合的累积不重复到达率，便于挑选最优组合。

## 工具 / 实现
本部分说明如何通过 SPSS 进行 TURF 分析，主要面向用研同学和动手能力较强的非用研同学。

首先需安装 SPSS 22.0 或以上版本（22.0 以下版本可能需安装 python 插件）。

TURF 分析所在位置：**分析 - 描述统计 - TURF 分析**。

1. 进入 TURF 分析页面后，选取要分析的所有题目（如所有媒体渠道的选项）；
2. "最大组合变量数"表示最多可采用多少个选项进行组合，可根据具体选项数量填写；
3. "要显示的组合数"可与最大组合变量数保持一致；
4. "正响应值"与多选题选中该选项的编码保持一致（如多选题中 1 表示选中、0 表示没选，则正响应值填 1）；
5. 点击确认按钮输出结果；

![](https://img14.360buyimg.com/ling/jfs/t1/59527/27/13109/320841/5da9656bE79d21855/9df5e022cdf72419.png)

6. 查看输出的"根据群组大小的最佳范围及次数"表格，其中"变数"列显示选项组合，"观察值的百分比"列显示累积不重合百分比。

![](https://img12.360buyimg.com/ling/jfs/t1/59062/36/13686/51259/5da96841Edc5c7b3d/de2365874a759344.png)

从"变数"列可看出 1-8 个选项组合时，不同数量的选项如何组合才能覆盖最多用户。例如选择 3 个选项的组合时，微信朋友圈 + 微信公众号 + 网购平台这个组合能达到最大覆盖率，覆盖用户比例 78.8%（对应"观察值的百分比"列）。

## 局限与误用
- **不能只看单项触达率排序**：触达率最高的几个项目直接叠加未必是最优组合，用户重合度高会被去重抵消（案例 1 中微博 + 微信朋友圈劣于微博 + 短视频 APP）。
- **潜在规模大 ≠ 贡献大**：潜在用户更多的产品若蚕食严重，对整体规模的净贡献反而更小（案例 2 产品 C）。需以"累积不重复增量"而非"单体规模"为判据。
