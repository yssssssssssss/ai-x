# 访谈提纲设计规范

## 目的与适用范围

本规范定义访谈提纲（含单人深访与焦点小组）的结构、内容构成、文字形式与执行边界的**必须遵守项**。适用于所有使用访谈法采集数据的研究项目，包括但不限于：

- 半结构式一对一深访 (SSI/DI)
- 线上/线下焦点小组 (FGD)
- CATI 电话访问提纲（需额外遵守场景适配原则）

方法本体"怎么做"见 `methods/toolbox/collection/interview-guide-design.md`；本文件讲"必须遵守什么"。

## 必须遵守项

### 一、提纲结构（四段式）

所有访谈提纲必须包含以下四段，顺序不可调换：

1. **开场**：向被访者介绍访谈流程和纪律，基本信息的自我介绍及简单破冰
2. **问题**：提纲的主体部分，将信息采集维度转变成用户语言（常规问题 + 深度追问 + 隐私问题）
3. **互动**：提纲的辅助部分，通过特定行为和任务激发被访者灵感、确认观点
4. **结语**：针对主体内容查漏补缺、向被访者致谢

### 二、开场规则

#### 单人访问 (DI)

**内容构成（按顺序）：**
- 问候语：简单问候，表达礼貌
- 自我介绍：项目/团队名称 + 访问员姓名及职能
- 调研内容：简述调研主要方向
- 安全承诺：个人信息绝对安全，反馈内容最小范围使用
- 用户价值："示弱"式破冰，凸显被访者专业性，不涉及购买推荐/强制操作
- 流程要求：相对安静环境，尽量不中断
- 致谢语：提前感谢参与
- 被访者介绍：请被访者简单介绍基本情况

**形式要求：**
- 问候语 + 自我介绍 + 调研内容：第一段，≤100 字，使用敬语
- 安全承诺：第二段，≤50 字，用词严谨正式
- 用户价值 + 流程要求：第三段，≤100 字，强调身份稀有/反馈指导产品
- 致谢语：≤30 字
- 被访者介绍：≤30 字

#### 焦点小组 (FGD)

在单人访问基础上的**差异项**：
- 自我介绍额外包含主持人生活/工作基本情况
- 安全承诺额外要求"请所有参会者共同保密会中内容"，≤100 字
- 流程要求额外说明"根据主持人要求按次序回答或自由顺序表达"
- 用户价值 + 流程要求：≤200 字
- 被访者介绍：逐个介绍姓名、生活、工作基本情况，与主持人介绍内容对等

### 三、问题规则

#### 通用规则（所有问题必须遵守）

| 原则 | 要求 | 禁止示例 |
|---|---|---|
| **通俗性** | 用大白话，避免专业术语 | "干预措施"→"改进措施"，"认知度"→"了解程度" |
| **中立性** | 无诱导、无倾向 | 禁止"你是否也认为本品效果很好？" |
| **清晰性** | 时间锚点必须明确，不用否定形式提问 | "最近"→"最近一个月内" |

#### 形式要求

- 每题题干 ≤30 字
- 题目理解和填答建议统一放至题目末尾括号中注明
- 题干关键词视觉加强（加粗/着色/下划线），每题 ≤2 个关键词
- 单人访问用单数第二人称，焦点小组用复数第二人称

#### 问题类型规范

| 类型 | 提问方向 | 提问维度 | 提问方式 | 数量上限 |
|---|---|---|---|---|
| **行为类** | 收集客观已发生的事实（访谈"地基"） | 行为内容/频率/时间/场景/方式 | 时空锚定法、过程分解法、5W1H | ≤5 题 |
| **态度类** | 认知评价/偏好/满意程度（"主观框架"） | 整体满意度/节点满意度/竞品对比/偏好排序/态度强度 | 打分法、形容词联想、事物联想、对比法 | ≤5 题（题干不出现态度词/倾向词） |
| **动机类** | 行为与态度背后的深层驱动力（"冰山之下"） | 决策触发点/选择排他性/需求底层逻辑/决策矛盾点 | 内在驱动力、情境投射、价值感知、价值关联 | ≤5 题（题干完全开放不举例） |
| **深度追问** | 应急即兴，实时深挖的"手术刀" | 模糊表述澄清/矛盾信息核验/极端案例深挖/潜台词挖掘 | 关键词解释、解释性复述 | 不限（临场决定） |
| **隐私问题** | 高敏感度问题 | 人口属性/消费属性/敏感行为/联系方式 | 第三人称投射法、区间选择法、出格行为常态化、不强制回答 | **≤2 题，必须放最后 10%** |

### 四、互动规则

#### 单人访问 — 5 类互动

| 互动类型 | 触发场景 | 操作话术 |
|---|---|---|
| **澄清确认** | 模糊、笼统表述 | "您刚刚说的 XX，具体指什么？""我理解您的意思是……对吗？" |
| **深度挖掘** | 行为/评价需层层递进 | "能不能举一件具体的事说明？""出现这种情况最主要的原因是什么？""还有其他补充吗？" |
| **情绪共情** | 负面感受、委屈、不满 | "我能理解这种感受，换作是我也会觉得困扰，您可以放心多说一点" |
| **控场纠偏** | 严重跑题 | "您说的这点很有意思，我们稍后再聊，先回到刚才 XX 相关的问题" |
| **留白鼓励** | 短暂沉默、不知如何作答 | "没关系，您慢慢想，想到什么都可以直接说" |

#### 焦点小组 — 6 类互动

| 互动类型 | 触发场景 | 操作话术 |
|---|---|---|
| **均等发言引导** | 有人长期不发言 | "刚刚几位都分享了想法，XX 先生/女士，您有没有不一样的感受？" |
| **观点碰撞激发** | 单一统一观点 | "现场所有人都是这个想法吗？有没有人存在不同看法？" |
| **分歧调和** | 组员争论、对立 | "大家的角度不一样，两种想法都有参考价值，我们分别听听双方的理由" |
| **信息补充** | 观点信息单薄 | "大家在生活里有没有遇到过类似场景，可以分享一下亲身经历" |
| **跑题收拢** | 偏离主题 | "这个话题很有价值，会后可以简单交流，我们先回到本次讨论的 XX 主题" |
| **总结复盘** | 单个话题结束 | "刚刚大家主要提到三点，有没有哪位想补充其他内容？" |

### 五、结语规则

#### 单人访问 — 5 步收尾

1. **信息补全确认**：用 3–5 分钟快速回溯 3–5 个关键信息点做交叉核验
2. **预留开放补充空间**："今天我们聊了很多内容，有没有什么相关的点是我刚才没问到，但你觉得很重要想补充的？"
3. **保密承诺二次强化**：再次明确匿名化处理、不对外泄露、原始录音按约定时间销毁
4. **回访与激励告知**：说明激励发放方式/到账时间；如有后续回访需求提前征得同意（≤10 分钟）
5. **正式致谢与离场**：真诚感谢，入户场景主动带走垃圾，不翻动被访者物品

#### 焦点小组 — 5 步收尾

1. **核心观点集体复盘**：2–3 分钟梳理共性核心观点，确认有无遗漏
2. **全员补充发言机会**："有没有哪位朋友还有 1 分钟以内的小想法想最后补充的？"
3. **匿名规则再次公示**：音视频仅用于内部分析、不对外传播、完全脱敏处理
4. **激励发放与离场引导**：统一说明礼品/酬劳领取流程，引导有序离场
5. **现场物料收尾**：回收便签/纸质材料，检查遗留物品，确认设备停止录制

## 推荐 vs 禁止做法

### 推荐

- 开放性原则：尽量使用开放式问题，给充分回答空间
- 逻辑递进原则：从易到难、从事实到观点，避免开头就抛敏感问题
- 场景适配原则：入户/现场深访可承载长时复杂内容；CATI 提纲 ≤10 分钟，避免视觉类问题
- 动机类问题鼓励被访者真实自由表达

### 禁止

- **概念抽象**：使用学术性/复合性/宏观的抽象名词
- **问题含糊**：题干缺少关键限制条件（时间/地点/频率/参照系/身份视角）
- **提问倾向**：题干包含价值判断/权威暗示/社会期许/情绪化形容词
- **双重含义**：一道题同时问两个不同问题却只允许一个答案
- **隐私问题前置**：隐私类问题放在访谈开头（必须放最后 10%）
- **深度追问带质疑语气**：触发被访者防御心理

## 检查清单

- [ ] 提纲包含完整四段式结构（开场→问题→互动→结语）
- [ ] 开场段包含安全承诺且字数达标
- [ ] 所有题干 ≤30 字，关键词加强 ≤2 个
- [ ] 行为/态度/动机各类问题 ≤5 题
- [ ] 隐私问题 ≤2 题且位于最后 10%
- [ ] 无术语/无诱导/无否定/无双重含义
- [ ] 互动话术覆盖核心场景（至少 5 类/6 类）
- [ ] 结语包含保密二次强化和激励告知
- [ ] CATI 提纲总时长 ≤10 分钟
