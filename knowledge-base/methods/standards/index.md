# methods/standards/ · 执行规范标准

> 内部「**必须遵守**」的研究规矩——权威规则，本库**单一出处**。
> 何时进：「我们组做 X 必须遵守什么」。
> 边界：`toolbox/` 讲「怎么做」（知识）；这里讲「必须遵守什么」（规则），由 toolbox 反向链接过来。
