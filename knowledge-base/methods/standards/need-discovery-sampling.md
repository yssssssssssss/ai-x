# 需求洞察的样本选择

> 内部「必须遵守」的权威规则。本规范在本库**单一出处**，其他文件引用而非复制。
> 落点：`methods/standards/`。边界：`toolbox/` 讲「怎么做」（知识）；本规范讲「需求洞察类项目挑用户必须遵守什么」（规则）。
> 同族规范：`methods/standards/sampling.md`（定量随机抽样，待建）——本规范聚焦定性需求洞察的样本选择。

## 适用范围

- **适用**：需求洞察、机会发现、概念探索类**定性研究**的用户招募与样本结构设计。
- **不适用**：定量调研——定量要求随机抽样、样本反映真实分布；本规范服务于"发现需求"而非"统计推断"。

## 必须遵守（MUST）

- 用户必须**有需求问题**且**愿意 + 善于表达**——不满足这两条的样本，访谈再多也产不出洞察。
- **不**追求样本的真实分布，明确按"是否能贡献洞察"来选用户。
- 样本必须**至少跨越**典型用户与极端用户两类，单一类别取样会系统性遗漏需求趋势信号。
- 招募说明书必须把"为什么找这类用户"写清楚——这是后续校验访谈对象是否被招对的依据。

## 推荐（SHOULD）

按下面四类组合来设计样本结构，每个项目根据研究问题挑 2–3 类：

### 1. 典型用户

> 一提起产品/服务，脑海中立刻浮现的形象（保健品→老年人；化妆品→年轻女性）。产品最核心的使用群体。

- **作用**：最快速了解行业/产品全貌，建立"主流使用者标准"作为对照基线。
- **何时一定要有**：研究刚启动、对行业还没基本认知时。

### 2. 极端用户（非主流用户，正态分布两端）

划分维度可以是技术水平、收入、年龄、规模占比等，按研究问题挑维度。

- **作用一**：放大那些在整体里看不见的细节差异。
- **作用二**：通过极端 vs 典型的行为差异，**发现未来需求趋势**——极端用户往往最早体现新需求。
- **作用三**：研究"落后型"极端用户（如老年人用移动互联网的障碍），等于把影响典型用户的因素拿放大镜看。

技术水平维度上有两类**重点关注**的极端用户：

- **专家型用户**：对产品深度了解、高频使用；他们今天的需求 ≈ 普通用户未来的需求；他们用产品的"变通技巧"是关键洞察来源。
- **新手用户**：用户成长流水线（非用户→新手→初级→高级→专家）的入口动力装置；研究他们 = 洞察"为什么进不来"以及（高频业务里）"为什么用几次就走"。

### 3. 趋势用户（容易引领消费趋势的人）

中国语境下四类典型趋势引领者：

| 人群 | 为什么是趋势引领者 |
|---|---|
| **大学生** | 教育期结束、自我意识觉醒、有家庭资助不愁生计、爱尝鲜、知识储备到位——典型新需求尝试者 |
| **北上广深都市外企白领** | 收入较高、技能水平要求高的职业、有个人时间、与国际接轨（接触趋势 + 获取国际产品的能力）——海淘最早就是这批人推动的 |
| **创意设计行业从业者** | 艺术/广告/建筑等高创意职业（自由职业者为典型），美学修养 + 反主流基因，对音乐/动漫等亚文化产品有超前预判 |
| **IT 从业人员** | "程序猿"，智商收入双高且年轻，技术零门槛，电子产品发展趋势的领先指标 |

### 4. 关联专家（不研究他们的需求，借他们的视角看需求）

> 关联专家不必是高学历或著作等身的教授。

- **核心**：通过专家的专业视角发现**目标用户自己说不清的潜在需求**。
- **示例**（手部护肤品）：手模、钢琴家、皮肤科 / 骨科专家、做手部护理和按摩的人、美甲专家——这些人不是核心目标用户，但对手部的保养有独到见解。
- **何时用**：目标用户难以自陈深层需求时；新品类、新场景探索时。

## 禁止（MUST NOT）

- 禁止把"愿意来 / 容易找到"作为样本选择的唯一标准——这会偏向时间多、表达欲强的边缘人群，不一定是有需求的人。
- 禁止只用典型用户做需求洞察——会系统性遗漏未来趋势信号。
- 禁止把"关联专家"的话当作"目标用户的需求陈述"——专家提供的是**视角**，不是**用户声音**。
- 禁止跨研究复用极端用户的划分维度而不重新论证——极端用户是相对维度，换研究问题就要换维度。

## 检查清单

- [ ] 项目的研究问题是否属于"需求洞察 / 机会发现 / 概念探索"？（若否，本规范不适用）
- [ ] 样本结构里是否同时有典型用户与极端用户？
- [ ] 极端用户的划分维度是否针对**这次研究问题**论证过？
- [ ] 是否需要趋势用户？（涉及"未来需求趋势"必须有）
- [ ] 是否需要关联专家？（目标用户难以自陈深层需求时必须考虑）
- [ ] 招募说明书有没有说清"为什么是这类人"？
- [ ] 表达能力是否做了筛选？（不善表达的样本要在筛选阶段剔除）
